| 第三 善哉                                                | 33. 好經                                                     |
| [尔时世尊]住舍卫城……                                     | 起緣於舍衛城。                                               |
| 尔时，沙睹罗巴天神等，其胜光偏照只园。黎明往诣世尊住处。 | 那時，在夜已深時，容色絕佳的眾多屬於沙睹羅巴眾天神使整個祇樹林發光後，去見世尊。 |
| 诣已礼敬世尊，立于一面。                                 | 抵達後，向世尊問訊後，在一旁站立。                           |
| 立于一面之天神，于世尊前，唱此欢喜偈曰：                 | 在一旁站立的一位天神在世尊的面前吟出這優陀那：               |
| 1友善哉布施                                              | 「親愛的先生！布施是好的，                                   |
| 2悭贪与放逸                                              | 以慳吝與放逸，                                               |
| 3如是不行施 4由知以明果                                  | 這樣布施不被施與，                                           |
| 5欲为功德者                                              | 希望福德者，                                                 |
| 6而行于布施                                              | 是施物的了知者。」                                           |
| 尔时，其他之天神，于世尊前，唱此欢喜 偈曰：              | 那時，另一位天神在世尊的面前吟出這優陀那：                   |
| 1善哉行布施                                              | 「親愛的先生！布施是好的，                                   |
| 2善哉贫困施                                              | 此外在少許中的布施也是好的，                                 |
| 3或人贫亦施                                              | 一些在少許中施與，                                           |
| 4或富不好施                                              | 一些在許多中不想要施與，                                     |
| 5贫困之施与                                              | 在少許中被施與的供養，                                       |
| 6功德计千倍                                              | 是等同千倍之量的。」                                         |
| 尔时，其他之天神，于世尊前，唱此欢喜偈曰：               | 那時，另一位天神在世尊的面前吟出這優陀那：                   |
| 1善哉行布施                                              | 「親愛的先生！布施是好的，                                   |
| 2贫困施善哉                                              | 在少許中的布施也是好的，                                     |
| 3依信施亦善                                              | 此外以信的布施也是好的，                                     |
| 4布施战同谓                                              | 他們說布施與戰爭是相同的，                                   |
| 5如少数善人 6以胜于多数                                  | 即使少許善的也打勝眾多的，                                   |
| 7若贫依信施                                              | 如果有信者即使施與少許，                                     |
| 8利他人安乐                                              | 就因為那樣他在來世是幸福者。」                               |
| 尔时，其他之天神，于世尊前，唱此欢喜偈曰：               | 那時，另一位天神在世尊的面前吟出這優陀那：                   |
| 1友善哉布施                                              | 「親愛的先生！布施是好的，                                   |
| 2贫困施善哉                                              | 在少許中的布施也是好的，                                     |
| 3依信施亦善                                              | 以信的布施也是好的，                                         |
| 4如法施亦善                                              | 此外如法所得的布施也是好的，                                 |
| 5精进于如法                                              | 凡施與如法所得的，                                           |
| 6以得布施者                                              | 以活力的努力所獲得的布施之人，                               |
| 7以越夜摩界 8卫多罗尼川                                  | 他超越閻摩王的[地獄]灰河後，                                 |
| 9而往于天界                                              | 死後來到天的住處。」                                         |
| 尔时，其他之天神，于世尊前，唱此欢喜偈曰：               | 那時，另一位天神在世尊的面前吟出這優陀那：                   |
| 1善哉行布施                                              | 「親愛的先生！布施是好的，                                   |
| 2贫困施善哉                                              | 在少許中的布施也是好的，                                     |
| 3依信施亦善                                              | 以信的布施也是好的，                                         |
| 4如法施亦善                                              | 如法所得的布施也是好的，                                     |
| 5辨别施亦善                                              | 此外簡別後的布施也是好的，                                   |
| 6辨别之施者 7善逝所赞叹                                  | 簡別後的布施被善逝稱讚，                                     |
| 8此世值供养                                              | 凡這裡在生命世界中應該被供養者，                             |
| 9行施有大果                                              | 在這些上的布施有大果，                                       |
| 10犹善田莳种                                             | 如種子被播種在良田中。」                                     |
| 尔时，其他之天神，于世尊前，唱此欢喜偈曰，               | 那時，另一位天神在世尊的面前吟出這優陀那：                   |
| 1善哉行布施                                              | 「親愛的先生！布施是好的，                                   |
| 2贫困施亦善                                              | 在少許中的布施也是好的，                                     |
| 3依信施亦善                                              | 以信的布施也是好的，                                         |
| 4如法施亦善                                              | 如法所得的布施也是好的，                                     |
| 5辨别施亦善                                              | 簡別後的布施也是好的，                                       |
| 8善哉求生活 6自制于有生                                  | 此外在生命類上[自我]抑制也是好的，                           |
| 7不害有生者                                              | 凡對活的生命類不惱害的行者，                                 |
| 9惧人之责难 10以致不为恶                                 | 以他人的斥責他們不作惡，                                     |
| 11如是赞惧者 12责不惧勇者                                | 在那裡他們讚賞膽小者而非英雄，                               |
| 13善人惧责难 14以不为恶事                                | 因為善人以害怕而不作惡。」                                   |
| 尔时，其他之天神，以此向世尊曰：                         | 那時，另一位天神在世尊的面前說這個：                         |
| ‘世尊!谁之偈为善说耶?’                                   | 「世尊！誰的被善說呢？」                                     |
| [世尊曰：]                                               |                                                              |
| 依汝等各各皆善说，                                       | 「一切你們的以法門被善說，                                   |
| 然则，且闻我[欢喜偈]：                                   | 此外，請你們也聽我的：                                       |
| 1依信之布施 2被受种种誉                                  | 『確實(羅馬拼音版)布施被許多方式稱讚，                       |
| 3法句施尤胜                                              | 但法句甚至比布施是更好的，                                   |
| 4其以前之世 5过去世善人                                  | 因為在以前與更早以前善人們，                                 |
| 6有智慧诸人 7皆行于涅盘                                  | 有慧者們甚至到達涅槃。」                                     |
