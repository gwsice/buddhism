| 第五 嫌责天                                                  | 35. 挑毛病經                                                 |
| 尔时，世尊住舍卫城只树给孤独园。                             | 有一次，世尊住在舍衛城祇樹林給孤獨園。                       |
| 尔时，甚多嫌责天神等，其胜光偏照只园，于黎明往诣世尊住处，   | 那時，在夜已深時，容色絕佳的眾多挑毛病天神使整個祇樹林發光後，去見世尊。 |
| 诣已，立于空中。                                             | 抵達後，站在空中。                                           |
| 立于空中之一天神，于世尊前，唱此偈曰：                       | 在空中站好後，一位挑毛病天神在世尊的面前說這偈頌：           |
| 1非如有自己                                                  | 「對自己是一種方式者，                                       |
| 2异而夸示人                                                  | 凡如果宣說另一種方式，                                       |
| 3如赌博诈术                                                  | 如詐欺後賭博者的，                                           |
| 5皆依于偷盗 4其人所受用                                      | 以偷盜為他的那個受用的。                                     |
| 6以语之所行                                                  | 凡確實會被作-確實應該說那個，                                |
| 7勿语虚无为                                                  | 凡不會被作-不應該說那個，                                    |
| 8虽语不为者                                                  | 對說不做的者們，                                             |
| 9贤者以知此                                                  | 賢智者們遍知。」                                             |
| 世尊：                                                       |                                                              |
| 1唯语此道迹                                                  | 「這非僅以說的，                                             |
| 2只是于听闻                                                  | 或單方面的以聽聞，                                           |
| 3不能随之行                                                  | 能夠隨進入，                                                 |
| 4此道迹坚固                                                  | 凡這個堅固的道跡，                                           |
| 5贤者依道迹 6以修于禅定                                      | 以該者明智者們、禪修者們，                                   |
| 7解脱魔之缚                                                  | 從魔的繫縛被釋放。                                           |
| 9如不云不作                                                  | 確實明智者們不做：                                           |
| 8贤者知世法                                                  | 知道世間法門後，                                             |
| 10贤者依智慧 11而入于涅盘                                    | 明智者們被完全智冷卻，                                       |
| 12以度世执着                                                 | 已度脫世間中執著。」                                         |
| 尔时，彼天神等，下来立于地上，头面顶礼世尊足，向世尊言：     | 那時，那些天神站到地上後，以頭落在世尊的腳上後對世尊說這個： |
| ‘世尊!是我等之罪、是我等之过。恰如愚者、迷者不善者，我等想： | 「大德！罪過征服如是愚的、如是愚昧的、如是不善的我們：       |
| 可如是责难世尊否?                                            | 凡我們認為世尊應該被攻擊。                                   |
| 愿世尊饶恕我等之罪，为未来不[再]犯。’                        | 大德！為了未來的自制，請世尊接受那些我們的罪過為罪過。」     |
| 时，世尊微笑。                                               | 那時，世尊顯露微笑。                                         |
| 时，彼天神等，更怒而升空。                                   | 那時，那些天神成為更加挑毛病者，升上空中。                   |
| 一天神于世尊前，唱此偈曰：                                   | 一位挑毛病天神在世尊的面前說這偈頌：                         |
| 1我等以谢罪                                                  | 「對懺悔過錯者們，                                           |
| 2而不受饶恕                                                  | 凡如果不接受者，                                             |
| 3内蓄恚怒重                                                  | 內部憤怒、嚴重的瞋恚，                                       |
| 4其人更结怨                                                  | 他到達敵意。」                                               |
| 5若无有罪过                                                  | 「如果不存在過錯，                                           |
| 6岂有此过失                                                  | 以及這裡沒有入歧途的，                                       |
| 7怨恚若不静                                                  | 如果不平息敵意，                                             |
| 8于此何有善                                                  | 這裡以誰是善的？」                                           |
| 9谁无有罪过                                                  | 「誰的過錯不存在？                                           |
| 10谁无有过失                                                 | 誰沒有入歧途的？                                             |
| 11谁不随失念                                                 | 誰不來到迷妄？                                               |
| 12谁贤常正念                                                 | 而誰是經常具念的明智者？」                                   |
| 世尊：                                                       |                                                              |
| 2于如来觉者                                                  | 「對如來、佛陀，                                             |
| 1怜愍诸有情                                                  | 一切生類的憐愍者，                                           |
| 3无有诸罪过                                                  | 他的過錯不存在，                                             |
| 4亦无有过失                                                  | 他沒有入歧途的，                                             |
| 5失念佛不堕                                                  | 他不走到迷妄，                                               |
| 6彼贤常正念                                                  | 他就是經常具念的明智者。」                                   |
| 7虽然以谢罪                                                  | 「對懺悔過錯者們，                                           |
| 8不得受宽恕                                                  | 凡如果不接受者，                                             |
| 9内蓄恚怒重                                                  | 內部憤怒、嚴重的瞋恚，                                       |
| 10其人更结怨                                                 | 他到達敵意，                                                 |
| 11若我不喜恚                                                 | 那敵意我不歡喜，                                             |
| 12即纳汝罪过                                                 | 我接受你們的罪過。」                                         |
