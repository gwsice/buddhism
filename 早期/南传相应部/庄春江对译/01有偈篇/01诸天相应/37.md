| 元亨寺                                                       | 庄春江                                                       |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 第七 会                                                      | 37. 集會經                                                   |
| 如是我闻。                                                   | 被我這麼聽聞：                                               |
| 尔时，世尊与诸阿罗汉之五百比丘，共住于释迦族迦毗罗城之大林。 | 有一次，世尊與約五百位比丘全部都是阿羅漢的大比丘僧團，共住在釋迦族的迦毘羅衛城大林中， |
| 十世界之多数诸天，亦为谒见世尊及比丘众而会集。               | 而從十個世間界的大部分天神，為了見世尊與比丘僧團已集合。     |
| 尔时，四净居之诸天如是思念：                                 | 那時，四位淨居天的天神想這個：                               |
| [今]世尊与诸阿罗汉之五百比丘共住于释迦族迦毗罗城之大林，     | 「這位世尊與約五百位比丘全部都是阿羅漢的大比丘僧團，共住在釋迦族的迦毘羅衛城大林中， |
| 十世界之多数诸天，为谒见世尊及比丘众而会集。                 | 而從十個世間界的大部分天神，為了見世尊與比丘僧團已集合，     |
| 我等亦诣世尊之处。                                           | 也讓我們去見世尊，                                           |
| 诣已，于世尊前，各唱其偈。                                   | 抵達後，在世尊的面前各自說偈頌。」                           |
| 如是彼等诸天，恰如有力人之伸屈臂腕，屈伸臂腕，如是没于净居天，现于世尊处。 | 那時，那些天神就猶如有力氣的男子伸直彎曲的手臂，或彎曲伸直的手臂，就像這樣在淨居天消失，出現在世尊的面前。 |
| 尔时，彼等诸天，礼敬世尊，立于一面，                         | 那時，那些天神向世尊問訊後，在一旁站立。                     |
| 立于一面之一天神，于世尊前唱偈曰：                           | 在一旁站立的一位天神在世尊的面前說這偈頌：                   |
| 1林中大集会                                                  | 「在叢林中的大集會，                                         |
| 2天众共会集                                                  | 天群已集合，                                                 |
| 4我等亦来赴 5此法之集会                                      | 我們來到這個法的集會，                                       |
| 3见无败僧伽                                                  | 見不敗的僧團。」                                             |
| 尔时，其他之天神，于世尊前，唱此偈曰：                       | 那時，另一位天神在世尊的面前說這偈頌：                       |
| 1于此比丘等                                                  | 「在那裡比丘們入定，                                         |
| 2自心静且直                                                  | 端正自己的心，                                               |
| 3如御执手纲                                                  | 如御車手握持韁繩(引導物)後，                                 |
| 4贤者守诸根                                                  | 賢智者們守護諸根。」                                         |
| 尔时，其他之天神，于世尊前，唱此偈曰：                       | 那時，另一位天神在世尊的面前說這偈頌：                       |
| 1割柱碎闩铨                                                  | 「切斷標柱、切斷橫木後，                                     |
| 2除栅离贪欲                                                  | 移除因陀羅柱後無擾動的，                                     |
| 3有眼者清净 4不污于游行                                      | 他們清淨地離垢地行，                                         |
| 5善调若幼象                                                  | 小龍象被有眼者善調御。」                                     |
| 尔时，其他之天神，唱此偈曰：                                 | 那時，另一位天神在世尊的面前說這偈頌：                       |
| 1谁归命于佛                                                  | 「凡任何已歸依佛者，                                         |
| 2不堕于恶趣                                                  | 他們將不去苦界之地，                                         |
| 3此人以舍身                                                  | 捨棄人身後，                                                 |
| 4应满于天集                                                  | 將使天身充滿。」                                             |
