| 元亨寺                                           | 庄春江                                                       |
| ------------------------------------------------ | ------------------------------------------------------------ |
| 第五 燃烧品                                      | 5.燃燒品                                                     |
| 第一 正在烧                                      | 41. 燃燒經                                                   |
| 如是我闻，                                       | 被我這麼聽聞：                                               |
| 尔时，世尊住舍卫城只树给孤独园。                 | 有一次，世尊住在舍衛城祇樹林給孤獨園。                       |
| 时，有天神于黎明时，其胜光偏照只园，往诣世尊处。 | 那時，在夜已深時，容色絕佳的某位天神使整個祇樹林發光後，去見世尊。 |
| 诣已，礼敬世尊立于一面。                         | 抵達後，向世尊問訊後，在一旁站立。                           |
| 立于一面之天神、于世尊前，唱此偈曰：             | 在一旁站立的那位天神在世尊的面前說這些偈頌：                 |
| 1由火焰燃家                                      | 「在家被燃燒時，                                             |
| 2搬出诸物品                                      | 凡取出器具，                                                 |
| 3惟只于此家                                      | 那是為了他的利益，                                           |
| 4是善不为烧                                      | 而非凡在那裡被燃燒[物]。                                     |
| 5如是于此世                                      | 這樣世間，                                                   |
| 6老死火炽烧                                      | 被老與死燃燒，                                               |
| 7布施而搬出                                      | 就應該以布施取出，                                           |
| 8施者为搬出                                      | 被施與的是善取出的。                                         |
| 9施乃得乐果                                      | 所施與的有樂果，                                             |
| 10不施无此果                                     | 未施與的它不是像那樣：                                       |
| 11依盗王者夺                                     | 盜賊國王們拿走，                                             |
| 12被火烧消灭                                     | 火燃燒、毀滅，                                               |
| 14死时无不舍                                     | 而在最後捨棄，                                               |
| 13如是富此身                                     | 包含財產的遺體。                                             |
| 15贤者以知此                                     | 有智慧者了知這樣後，                                         |
| 16享富且善施                                     | 應該受用及應該施與，                                         |
| 17从力用且施                                     | 如其能力地施與及受用後，                                     |
| 18行天无难责                                     | 無過失地到達天界處。」                                       |
