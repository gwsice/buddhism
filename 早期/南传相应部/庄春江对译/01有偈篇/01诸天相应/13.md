| 第三 无有如子者                      | 13. 沒有等同兒子者經                     |
|                                      | 起源於舍衛城。                           |
| 立于一面之天神，于世尊前，而唱偈曰： | 在一旁站立的那位天神在世尊面前說這偈頌： |
| 1可爱莫若子                          | 「沒有等同對兒子的愛，                   |
| 2致富莫若牛                          | 沒有等同牛之財，                         |
| 3光明莫若阳                          | 沒有等同太陽的光明，                     |
| 4海为最上湖                          | 大海是池湖中最勝的。」                   |
| [世尊：]                             |                                          |
| 1可爱莫若己                          | 「沒有等同對自我的愛，                   |
| 2致富莫若谷                          | 沒有等同穀物之財，                       |
| 3光明莫若慧                          | 沒有等同慧的光明，                       |
| 4雨为最上湖                          | 雨是池湖中最勝的。」                     |
