| 第二 欢喜园品                                                | 2.歡喜園品                                                   |
| 第一 欢喜园                                                  | 11. 歡喜園經                                                 |
| 如是我闻。                                                   | 被我這麼聽聞：                                               |
| 尔时，世尊住舍卫城只树给孤独园。                             | 有一次，世尊住在舍衛城祇樹林給孤獨園。                       |
| 于此，世尊以‘比丘等!’言诸比丘：                              | 在那裡，世尊召喚比丘們：「比丘們！」                         |
| ‘世尊!’彼等诸比丘奉答世尊。                                  | 「尊師！」那些比丘回答世尊。                                 |
| 世尊如是宣说：                                               | 世尊說這個：                                                 |
| ‘诸比丘!昔有属于三十三天之天神，于欢喜园、为天女群侍随围绕， | 「比丘們！從前，某位三十三天眾的天神在歡喜園中被天女眾圍繞， |
| 给与五欲具足，                                               | 當賦有、擁有天的五種欲自娛時，                               |
| 尔时唱此偈曰：                                               | 那時候，他說了這偈頌：                                       |
| 4不知此快乐                                                  | 『他們不了知樂：                                             |
| 3如非见此园                                                  | 凡沒見過歡喜園者，                                           |
| 2天神之住家                                                  | 是天人們的住所：                                             |
| 1享誉三三天                                                  | 三十三天的有名聲者。』                                       |
| 诸比丘!如是说已，有天神以偈答彼天神曰．                      | 比丘們！在這麼說時，某位天神以偈頌回應那位天神：             |
| 1汝为愚痴者                                                  | 『愚者！你不了知，                                           |
| 2不知如来语                                                  | 關於阿羅漢們的言語：                                         |
| 3诸行实无常                                                  | 一切行[確實]是無常的，                                       |
| 4乃是生灭性                                                  | 是生起與消散法的，                                           |
| 5生者必有灭                                                  | 生起後被滅，                                                 |
| 6寂此正悦乐                                                  | 它們的寂滅是樂。』                                           |
