| 第十 三弥提                                                  | 20. 三弥提经                                                 |
| 如是我闻，                                                   | 被我這麼聽聞：                                               |
| 尔时，世尊住王舍城温泉精舍。                                 | 有一次，世尊住在王舍城溫泉園。                               |
| 尔时，尊者三弥提，为洗浴身体，黎明即起，来至温泉。           | 那時，尊者三彌提在破曉時起來後，前往溫泉灌洗身體。           |
| 温泉洗身已，起身着一衣，站立以待身干。                       | 在溫泉灌洗身體後起來後，單衣站立弄乾著身體。                 |
| 尔时，有天神于夜将晓，以其胜光偏照温泉林，而往诣尊者处，     | 那時，在夜已深時，容色絕佳的某位天神使整個溫泉園發光後，去見尊者三彌提。 |
| 诣已立于空中，以偈告尊者三弥提曰：                           | 抵達後，成為空中站立者，以偈頌對尊者三彌提說：               |
| 1比丘不享乐2精进以行乞                                       | 「比丘！不享受後你乞食，                                     |
| 4享乐勿行乞                                                  | 享受後你不乞食，                                             |
| 3比丘行享乐                                                  | 比丘！享受後請你乞食，                                       |
| 5青春盛壮时6勿为徒空费                                       | 不要時間離開你。」                                           |
| 尊者：                                                       |                                                              |
| 1汝时我不知                                                  | 「時間我不知道，                                             |
| 2为时无隐显                                                  | 被隱藏的時間不被看見，                                       |
| 3如是之故我4行乞不享乐                                       | 因此不享受後我乞食，                                         |
| 5时之以舍我6是即为畏怖                                       | 不要時間離開我。」                                           |
| 尔时，天神降立于地上，言尊者三弥提曰：                       | 那時，那位天神站到地上後對尊者三彌提說這個：                 |
| ‘比丘!汝具青少年壮，肤美发黑，摒弃人生最盛期之欲乐而出家。   | 「比丘！你年輕出家，黑髮的青年，具備青春的幸福，在人生初期，不在欲中娛樂， |
| 比丘!享乐人之五欲!勿舍现实[乐]，勿逐时要之物。’              | 比丘！你要在人之欲中享受，不要捨棄直接可見的後追逐時間的。」 |
| ‘友!我舍现实乐，不逐时要物。                                 | 「朋友！我沒捨棄直接可見的後追逐時間的，朋友！但我捨棄時間的後追逐直接可見的， |
| 友!何以故，世尊宣说‘时要之欲乐，不安苦多，于此有大祸。       | 朋友！因為欲被世尊說是時間的、多苦的、多絕望的，在這裡有更多的過患。 |
| 此[圣]法乃现实，非隔时，得言来看，--可导[涅盘]，智者应自证知。’ | 這個法是直接可見的、即時的、請你來看的、能引導的、應該被智者各自經驗的。」 |
| ‘比丘!如何世尊宣说：‘时要之欲乐，苦多不安，于此有大祸，      | 「比丘！但怎樣是欲被世尊說是時間的、多苦的、多絕望的，在這裡有更多的過患？ |
| 此[圣]法是现实，非隔时，得言来看，可导[涅盘]，智者应自证知耶?’ | 怎樣是這個法是直接可見的、即時的、請你來看的、能引導的、應該被智者各自經驗的？」 |
| ‘友!我乃出家未久，是新[学]人初参者，不能广说此法与律，       | 「朋友！我是新人、出家不久者、剛來這法律者，我不能詳細地講述它， |
| 彼世尊，应供，正等觉者，今[住]温泉精舍，此义诣问彼世尊，尚请汝如世尊所说受持。’ | 那位世尊、阿羅漢、遍正覺者住在這王舍城溫泉園，你去見世尊後問這個道理，你應該如世尊為你解說那樣憶持它。」 |
| ‘比丘!彼世尊有大威力，为诸天所围绕、我等不易亲近。           | 「比丘！去見那位世尊不易被我們做，因為被其它大影響力的諸天圍繞。 |
| 比丘!汝若以此义诣问世尊，我等为听法应得亲近。’               | 比丘！如果你去見那位世尊後問這個道理，我們也會為了聽法而來到。」 |
| 尊者三弥提答其天神：‘友!唯诺。’即诣世尊住处，                | 「是的，朋友！。」尊者三彌提回答那位天神後，去見世尊。       |
| 诣已，礼敬世尊坐于一面。坐于一面之尊者三弥提，敬自世尊曰：   | 抵達後，向世尊問訊後，在一旁坐下。在一旁坐下的尊者三彌提對世尊說這個： |
| ‘世尊!我为洗浴身体，黎明即起，去至温泉。                     | 「大德！這裡，我在破曉時起來後，到溫泉灌洗身體。             |
| 于温泉洗身已，起身出着一衣，站立以待身干。                   | 在溫泉灌洗身體後起來後，單衣站立弄乾著身體。                 |
| 世尊，尔时有天神、于夜将晓，以其胜光偏照温泉林，来立于空中，以偈语我曰． | 那時，在夜已深時，容色絕佳的某位天神使整個溫泉園發光後，來見我。抵達後，站在空中以偈頌對我說： |
| 1比丘不享乐2精进以行乞                                       | 『比丘！不享受後你乞食，                                     |
| 4享乐勿行乞                                                  | 享受後你不乞食，                                             |
| 3比丘行享乐                                                  | 比丘！享受後請你乞食，                                       |
| 5青春盛壮时6勿为徒空费                                       | 不要時間離開你。』                                           |
| 世尊!如是言已，我以偈答天神曰：                              | 大德！在這麼說時，我以偈回應那位天神：                       |
| 1汝时我不知                                                  | 『時間我不知道，                                             |
| 2为时无隐显                                                  | 被隱藏的時間不被看見，                                       |
| 3如是之故我4行乞不享乐                                       | 因此不享受後我乞食，                                         |
| 5摒舍时之我6是即为恐怖                                       | 不要時間離開我。』                                           |
| 世尊!尔时，天神降立于地上，以此言我曰，                      | 大德！那時，那位天神站到地上對我說這個：                     |
| ‘比丘!汝具青少年壮，肤美发黑，勿乐人生最盛期之欲乐而出家。   | 『比丘！你年輕出家，黑髮的青年，具備青春的幸福，在人生初期，不在欲中娛樂， |
| 比丘!享乐人之[五]欲!勿舍现实[乐]，勿逐时要物。’              | 比丘！你要在人之欲中享受，不要捨棄直接可見的後追逐時間的。』 |
| 世尊!如是言已，我如是言天神曰：                              | 大德！在這麼說時，我這麼回答那位天神：                       |
| ‘友!我舍现实[乐]，不逐时要物。                               | 『朋友！我沒捨棄直接可見的後追逐時間的，                     |
| 友!我舍时要物，以逐现实[乐]。                                | 朋友！但我捨棄時間的後追逐直接可見的，                       |
| 友!何以故，世尊宣说：‘时要之欲乐，不安苦多，于此有大祸。     | 朋友！因為欲被世尊說是時間的、多苦的、多絕望的，在這裡有更多的過患。 |
| 此[圣]法乃现实，不隔时，得言来看，可导[涅盘]，智者应自证知。’ | 這個法是直接可見的、即時的、請你來看的、能引導的、應該被智者各自經驗的。』 |
| 世尊!言已!天神如是言我曰：                                   | 大德！在這麼說時，那位天神對我說這個：                       |
| ‘比丘!如何世尊宣说：‘乐欲是时要之物，不安苦多，于此有大祸。  | 『比丘！但怎樣是欲被世尊說是時間的、多苦的、多絕望的，在這裡有更多的過患？ |
| 此[圣]法是现实、非隔时，得言来看，可导[涅盘]，智者应自证知’耶?’ | 怎樣是這個法是直接可見的、即時的、請你來看的、能引導的、應該被智者各自經驗的？』 |
| 世尊!如是言已，我语天神曰：                                  | 大德！在這麼說時，我這麼回答那位天神：                       |
| ‘友!我出家未久，是新[学]人初参者，不能广说此法与律，         | 『朋友！我是新人、出家不久者、剛來這法律者，我不能詳細地講述它， |
| 彼世尊，应供，正等觉者，[今]住温泉精舍，                     | 那位世尊、阿羅漢、遍正覺者住在這王舍城溫泉園，               |
| 此义诣问彼世尊，尚请汝如世尊所说受持。’                      | 你去見世尊後問這個道理，你應該如世尊為你解說那樣憶持它。』   |
| 世尊!如是说已，天神语我曰：‘比丘!彼世尊有大威力，为诸天所围绕，我等不易亲近。 | 大德！在這麼說時，那位天神對我說這個：『比丘！去見那位世尊不易被我們做，因為被其它大影響力的諸天圍繞， |
| 比丘!汝若以此义诸问世尊，我等为听法应得亲近。’               | 比丘！如果你去見世尊後問這個道理，我們也會為了聽法而來到。』 |
| 世尊!其天神之语，若为真实，其天神当不远。’                   | 大德！如果那位天神的言語是真實的，那位天神就在這裡不遠處。」 |
| 如是言已，天神语尊者三弥提曰：                               | 在這麼說時，那位天神對尊者三彌提說這個：                     |
| ‘比丘!问!比丘等!问!我来矣!’                                  | 「比丘！請你問，比丘！請你問，我已到達。」                   |
| 尔时世尊以偈语天神曰：                                       | 那時，世尊以偈頌對那位天神說：                               |
| 1知名表面者                                                  | 「能被告知的有想眾生，                                       |
| 2唯执是显名                                                  | 在能被告知的上被建立，                                       |
| 3不知名真义                                                  | 不遍知能被告知的後，                                         |
| 4死魔系缚去                                                  | 來到死神的軛。                                               |
| 5了知名真义                                                  | 但遍知能被告知的後，                                         |
| 6不思语说者                                                  | 不思量告知者，                                               |
| 7其人不如此                                                  | 因為對他『那個不存在』：                                     |
| 8依此人无罪                                                  | 以那個依能說他的對他不存在。                                 |
| 9如果汝夜叉10知如是人言                                      | 如果你了知，請你說，夜叉！」                                 |
| ‘不然!世尊，我不知所略说之意义。                             | 「大德！我對這個被世尊簡要地說的，不詳細地了知義理，         |
| 善哉!世尊，得请说世尊所略说者令我能得广知其义耶!’            | 大德！請世尊能為我說明，依之我能詳細地了知這被世尊簡要地說的義理，那就好了！」 |
| 1我胜等或劣                                                  | 「相等的或者殊勝的或卑劣的：                                 |
| 2人思依此争                                                  | 凡他認為-他以那個會爭論，                                    |
| 3此等三种别4若心不动者                                       | 在三種上成為不動搖者，                                       |
| 5其人无胜劣                                                  | 他不想『相同的殊勝的』。                                     |
| 6如果汝夜叉7知如是人言                                       | 如果你了知，請你說，夜叉！」                                 |
| ‘不然!世尊，我不知世尊所略说之义。                           | 「大德！我對這個被世尊簡要地說的，也不詳細地了知義理，       |
| 善哉!世尊，得请说世尊所略说者，令我能得广知其义耶!’          | 請世尊能為我說明，依之我能詳細地了知這被世尊簡要地說的義理，那就好了！」 |
| 1拂思勿行慢                                                  | 「捨斷名稱他不到達勝慢，                                     |
| 2断世名色爱                                                  | 在這裡切斷在名色上的渴愛，                                   |
| 3不动绝结缚4无欲之彼者                                       | 那位繫結已被切斷、無苦惱、無願望者，                         |
| 7虽求人天迹8亦无有所得                                       | 當遍求時他們沒得到：                                         |
| 5后世于天界                                                  | 諸天、人們在此世或在他世，                                   |
| 6于所有住处                                                  | 在天界或在一切住處。                                         |
| 9如果汝夜叉10知如是人言                                      | 如果你了知，請你說，夜叉！」                                 |
| 世尊，我如是知世尊广宣所略说之意义。                         | 「大德！我對這個被世尊簡要地說的，這樣詳細地了知義理：       |
| 1于所有世界                                                  | 『世間中任何事物，                                           |
| 3勿为恶业行2依语与心身                                       | 惡的不應該被身、語、意作，                                   |
| 4离欲正念心                                                  | 有念正知地捨斷諸欲後，                                       |
| 6即无受痛苦5无有伴利者                                       | 不應該實行苦的、伴隨無利益的。』」                           |
