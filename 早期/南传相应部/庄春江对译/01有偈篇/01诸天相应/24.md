| 第四 制止心 | 24. 意的制止經           |
| 天神：      |                          |
| 1心所制止者 | 「應該從一一處制止意，   |
| 2不行心苦事 | 從該一一處苦不來到他，   |
| 3若总制止心 | 如果他從一切制止意，     |
| 4心脱一切苦 | 他從一切苦被釋放。[」]   |
| 世尊：      |                          |
| 1非总制止心 | 「不應該從一切制止意：   |
| 2非制自制心 | 非來到自我抑制的意，     |
| 3正于起恶处 | 從惡的一一處，           |
| 4应制此之心 | 從該一一處應該制止意。」 |
