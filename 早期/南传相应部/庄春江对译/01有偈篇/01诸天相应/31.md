| 第四 沙睹罗巴天群品                      | 4.沙睹羅巴眾品                                     |
| 第一 与善人为伴                          | 31. 與善人們經                                     |
| 如是我闻。                               | 被我這麼聽聞：                                     |
| 尔时世尊住舍卫城只树给孤独园。           | 有一次，世尊住在舍衛城祇樹林給孤獨園。             |
| 尔时甚多沙睹罗巴天神等，                 | 那時，在夜已深時，容色絕佳的眾多屬於沙睹羅巴眾天神 |
| 其胜光偏照只园。                         | 使整個祇樹林發光後，                               |
| 于黎明往诣世尊处，                       | 去見世尊。                                         |
| 诣已礼敬世尊，立于一面。                 | 抵達後，向世尊問訊後，在一旁站立。                 |
| 立于一面之一天神，于世尊前，唱此偈曰：   | 在一旁站立的一位天神在世尊的面前說這偈頌：         |
| 1唯与善人坐                              | 「只應該與善人們結交，                             |
| 2唯与善人交                              | 應該與善人們作親密交往，                           |
| 3知善之正法                              | 了知善的正法後，                                   |
| 4为善不为恶                              | 成為更好的、非惡的。」                             |
| 尔时，其他天神，于世尊前，亦唱此偈曰：   | 那時，另一位天神在世尊的面前說這偈頌：             |
| 1唯与善人坐                              | 「只應該與善人們結交，                             |
| 2唯与善人交                              | 應該與善人們作親密交往，                           |
| 3知善之正法                              | 了知善的正法後，                                   |
| 5而得之智慧 4依他不能得                  | 慧被得到而非從其他者。」                           |
| 尔时，其他之天神，于世尊前，亦唱此偈曰： | 那時，另一位天神在世尊的面前說這偈頌：             |
| 1唯与善人坐                              | 「只應該與善人們結交，                             |
| 2唯与善人交                              | 應該就與善人們作親密交往，                         |
| 3知善之正法                              | 了知善的正法後，                                   |
| 4悲中无有悲                              | 在憂愁中不憂愁。」                                 |
| 尔时，其他之天神，于世尊前，亦唱此偈曰： | 那時，另一位天神在世尊的面前說這偈頌：             |
| 1唯与善人坐                              | 「只應該與善人們結交，                             |
| 2唯与善人交                              | 應該就與善人們作親密交往，                         |
| 3知善之正法                              | 了知善的正法後，                                   |
| 4辉耀亲族中                              | 在親族中輝耀。」                                   |
| 尔时，其他之天神，于世尊前，亦唱此偈曰： | 那時，另一位天神在世尊的面前說這偈頌：             |
| 1唯与善人坐                              | 「只應該與善人們結交，                             |
| 2唯与善人交                              | 應該就與善人們作親密交往，                         |
| 3知善之正法                              | 了知善的正法後，                                   |
| 4人人行善趣                              | 眾生到善趣。」                                     |
| 尔时，其他之天神，于世尊前，亦唱此偈曰： | 那時，另一位天神在世尊的面前說這偈頌：             |
| 1唯与善人坐                              | 「只應該與善人們結交，                             |
| 2唯与善人交                              | 應該就與善人們作親密交往，                         |
| 3知善之正法                              | 了知善的正法後，                                   |
| 4人人有幸福                              | 眾生經常地住立。」                                 |
| 尔时，其他之天神，以此向世尊曰：         | 那時，另一位天神對世尊說這個：                     |
| ‘世尊!谁之[偈]为善说耶?’                 | 「世尊！誰的被善說呢？」                           |
| [世尊曰：]                               |                                                    |
| 依汝等各各之善说。                       | 「一切你們的以法門被善說，                         |
| 然则，亦闻我偈：                         | 此外，請你們也聽我的：                             |
| 1唯与善人坐                              | 『應該就與善人們結交，                             |
| 2唯与善人交                              | 應該就與善人們作親密交往，                         |
| 3知善之正法                              | 了知善的正法後，                                   |
| 4当脱一切苦                              | 從一切苦被釋放。』」                               |
