| 第二 悭贪                                                  | 32. 慳吝經                                                   |
| 尔时世尊住舍卫城只树给孤独园。                             | 有一次，世尊住在舍衛城祇樹林給孤獨園。                       |
| 尔时甚多沙睹罗巴天神等，其胜光偏照只园。于黎明往诣世尊处， | 那時，在夜已深時，容色絕佳的眾多屬於沙睹羅巴眾天神使整個祇樹林發光後，去見世尊。 |
| 诣已礼敬世尊，立于一面。                                   | 抵達後，向世尊問訊後，在一旁站立。                           |
| 立于一面之一天神，于世尊前，唱此偈曰：                     | 在一旁站立的一位天神在世尊的面前說這偈頌：                   |
| 1依悭贪放逸                                                | 「以慳吝與放逸，                                             |
| 2如是不布施                                                | 這樣布施不被施與，                                           |
| 3如是不布施 4由欲功德者                                    | 希望福德者，                                                 |
| 5而行于布施                                                | 是施物的了知者。」                                           |
| 尔时，其他之天神，于世尊前，唱此偈曰：                     | 那時，另一位天神在世尊的面前說這些偈頌：                     |
| 1悭贪畏布施 2畏彼不施与                                    | 「就凡害怕的慳吝者不施與，                                   |
| 3悭贪所恐怖                                                | 就成為那位不施與者的害怕：                                   |
| 4乃是饥与渴                                                | 飢餓與口渴，                                                 |
| 6必然以触此 5愚人此世后                                    | 凡慳吝者害怕的，那位愚者就接觸，在這個世間與下一個中。       |
| 7是故灭悭贪                                                | 因此調伏慳吝後，                                             |
| 8布施胜垢秽                                                | 征服垢者應該施與布施，                                       |
| 9功德实人人 10后世之浅渡                                   | 福德在其他世間(來世)中，是有生命者們的依止。」               |
| 尔时，其他之天神，于世尊前唱此偈曰：                       | 那時，另一位天神在世尊的面前說這些偈頌：                     |
| 3灭中有不灭                                                | 「他們在死者中不死，                                         |
| 1善导旷野旅                                                | 如道路的同行者，                                             |
| 2分与贫困者                                                | 凡他們在少許中施與者，                                       |
| 4此为永久法                                                | 這是永遠的法。                                               |
| 5有人与贫困                                                | 一些在少許中施與，                                           |
| 6或人富不与                                                | 一些在許多中不想要施與，                                     |
| 7施与贫困者                                                | 在少許中被施與的供養，                                       |
| 8功德计千倍                                                | 是等同千倍之量的。」                                         |
| 尔时，其他之天神，于世尊前，唱此偈曰：                     | 那時，另一位天神在世尊的面前說這些偈頌：                     |
| 1难与而与之                                                | 「對難施與的施與者們，                                       |
| 2难为而为者                                                | 難做事務(業)的作者們：                                       |
| 3不善人难愦                                                | 不善者們不模仿，                                             |
| 4从善人法难                                                | 正法是難隨行的。                                             |
| 5是故善与恶                                                | 因此善者不善者們，                                           |
| 6后世趣所具                                                | 從這裡有種種趣處，                                           |
| 7恶人行地狱                                                | 不善者們去地獄，                                             |
| 8善人趣天堂                                                | 善者們有天界趣處。」                                         |
| 尔时，其他之天神，以此向世尊曰：                           | 那時，另一位天神在世尊的面前說這個：                         |
| ‘世尊!谁之偈为善说耶?’                                     | 「世尊！誰的被善說呢？」                                     |
| [世尊曰：]                                                 |                                                              |
| 依汝等各各皆善说，然则，以闻我偈：                         | 「一切你們的以法門被善說，此外，請你們也聽我的：             |
| 1虽以拾落穗                                                | 『凡拾落穗者如果也行法，                                     |
| 2虽以拾落穗 3贫困以行施 4是行善业法                        | 如果扶養妻子者行在少許中施與的，                             |
| 5虽千之供牺 6百千之供牺                                    | 百千個的供千犧牲者，                                         |
| 7斯施犹不值 8十六分之一                                    | 他們也不值得像那樣的十六分之一。』」                         |
| 尔时，其他之天神，向世尊说此偈曰：                         | 那時，另一位天神以偈頌對世尊說這個：                         |
| 1如何丰供牺                                                | 「為何這廣大的牲祭，                                         |
| 2不值于正施                                                | 不等同地來到所施的價值呢？                                   |
| 3千之供牺者 4百千之供牺                                    | 為何百千個的供千犧牲者，                                     |
| 5此牺何不值 6十六分之一                                    | 他們也不值得像那樣的十六分之一呢？」                         |
| 尔时，世尊以偈语其天神曰：                                 |                                                              |
| 1或人之错误 2立施于三业                                    | 「因為一些施與被建立在不正的上，                             |
| 3伤杀又恼施                                                | 切斷、殺害及使之憂愁後，                                     |
| 4泪污杀害施                                                | 那些淚滿面的、有懲罰的供養，                                 |
| 5不值于正施                                                | 不等同地來到所施的價值。                                     |
| 6虽此千供牺 7百千之供牺                                    | 這樣百千個的供千犧牲者，                                     |
| 8斯施犹不值 9十六分之一                                    | 他們也不值得像那樣的十六分之一。」                           |
