| 第六 信                                                    | 36. 信經                                                     |
| 尔时，世尊住舍卫城只树给孤独园。                           | 有一次，世尊住在舍衛城祇樹林給孤獨園。                       |
| 尔时，甚多沙睹罗巴天神等，其胜光偏照只园，黎明往诣世尊处。 | 那時，在夜已深時，容色絕佳的眾多屬於沙睹羅巴眾天神使整個祇樹林發光後，去見世尊。 |
| 诣已，礼敬世尊，立于一面。                                 | 抵達後，向世尊問訊後，在一旁站立。                           |
| 立于一面之一天神，于世尊前，喝此偈曰：                     | 在一旁站立的一位天神在世尊的面前說這偈頌：                   |
| 1信仰人之侣                                                | 「信是男子的伴侶，                                           |
| 2若人不无信 3而在于此世                                    | 如果無信則不住立，                                           |
| 4有誉及称赞                                                | 從那裡會有名聲與稱譽，                                       |
| 5死后生天界                                                | 捨棄遺骸後他到天界。」                                       |
| 天神：                                                     | 那時，另一位天神在世尊的面前說這偈頌：                       |
| 1舍慢离忿恨                                                | 「應該捨棄憤怒應該放棄慢，                                   |
| 2超越一切缚                                                | 應該超越一切結，                                             |
| 3不执名色者                                                | 他在名色上不執著，                                           |
| 4无何所结缚                                                | 苦不降臨無所有者。」                                         |
| 5鲁钝无智慧                                                | 「愚者們、劣智慧的人們，                                     |
| 6人人耽放逸                                                | 從事放逸，                                                   |
| 8贤守不放逸                                                | 而有智慧者對不放逸，                                         |
| 7如护最胜富                                                | 如最上的財產般地保護。」                                     |
| 9莫耽于放逸                                                | 「你們應該不要從事放逸，                                     |
| 10莫交爱欲乐                                               | 不要有欲之喜樂的親密交往，                                   |
| 11禅思不放逸                                               | 因為不放逸的修禪者，                                         |
| 12以达最胜乐                                               | 到達最高的[廣大的]樂。」                                     |
