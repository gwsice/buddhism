| 第八 岩石之破片                                              | 38. 碎石片經                                                 |
| 如是我闻。                                                   | 被我這麼聽聞：                                               |
| 尔时，世尊住王舍城摩达屈支之鹿野苑。                         | 有一次，世尊住在王舍城嘛瘩姑七的鹿林。                       |
| 尔时，世尊之足，为岩石破片所伤。                             | 當時，世尊的腳被碎石片所傷，                                 |
| 世尊甚感痛楚；身之所受非常强烈，心中颇有不快。               | 世尊的強烈感受轉起：苦的、激烈的、猛烈的、強烈的、不愉快的、不合意的身體的感受， |
| 世尊以正心正念之忍耐，而不为之所恼。                         | 世尊具念地、正知地忍受它，不被惱害著。                       |
| 时，世尊将僧伽梨叠为四重，正心正念，两足相叠，以作右胁师子卧。 | 那時，世尊摺大衣成四折後，[左]腳放在[右]腳上後，具念正知地以右脅作獅子臥。 |
| 时，七百沙睹罗巴天神等，夜中往诣世尊处，其胜光偏照摩达屈支。 | 那時，在夜已深時，容色絕佳的七百位屬於沙睹羅巴眾天神使整個嘛瘩姑七發光後，去見世尊。 |
| 诣已，礼敬世尊，立于一面。                                   | 抵達後，向世尊問訊後，在一旁站立。                           |

| 正心正念得堪忍，是为依其龙象之性。’                          | 以如龍象的行為模式具念地、正知地忍受它，                     |
|                                                              | 不被惱害著。」                                               |
