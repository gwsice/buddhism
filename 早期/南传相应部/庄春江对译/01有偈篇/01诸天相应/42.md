| 第二 以与何 | 42. 施與什麼經             |
| 天神：      |                            |
| 1与何与力耶 | 「施與什麼者是施與力氣者？ |
| 2与何美貌耶 | 施與什麼者是施與容色者？   |
| 3与何安乐耶 | 施與什麼者是施與安樂者？   |
| 4与何与眼耶 | 施與什麼者是施與眼睛者？   |
| 5与何一切耶 | 以及誰是施與一切者？       |
| 6问之对我说 | 請你告知我所問的。」       |
| 世尊：      |                            |
| 1与食物与力 | 「施與食物者是施與力氣者， |
| 2与衣与美貌 | 施與衣服者是施與容色者，   |
| 3与车乘与乐 | 施與車乘者是施與安樂者，   |
| 4与灯明与眼 | 施與燈火者是施與眼睛者。   |
| 6是为与一切 | 以及他是施與一切者：       |
| 5与住处人者 | 凡施與住房者，             |
| 8是与不死人 | 但他是施與不死者：         |
| 7教与正法者 | 凡教誡法者。」             |
