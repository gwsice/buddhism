| 第四 非然                                                    | 34. 沒有經                                                   |
| 尔时，世尊，住舍卫城只树给孤独园。                           | 有一次，世尊住在舍衛城祇樹林給孤獨園。                       |
| 尔时，甚多沙睹罗巴天神等，其胜光偏照只园、于黎明往诣世尊住处。 | 那時，在夜已深時，容色絕佳的眾多屬於沙睹羅巴眾天神使整個祇樹林發光後，去見世尊。 |
| 诣已礼敬世尊，立于一面，                                     | 抵達後，向世尊問訊後，在一旁站立。                           |
| 立于一面之天神，于世尊前，唱此偈曰：                         | 在一旁站立的一位天神在世尊的面前說這偈頌：                   |
| 1人人中爱欲 2乃非是常住                                      | 「在人們中諸欲沒有常的，                                     |
| 3于此有爱乐 4人人被系缚                                      | 在這裡有諸能被欲求的-在那些上被繫縛，                        |
| 5于此以放逸 6而无作归来                                      | 在那些上放逸者是不再來[涅槃-㊟]者：                          |
| 7人人死魔领 8如是不归来                                      | 男子是從死亡領域不再來者。」                                 |
| 9祸从欲望生 10苦恼从欲生                                     | 「痛苦是意欲生的、苦是意欲生的，                             |
| 11以调伏欲望 12是则祸调伏                                    | 以意欲的調伏有痛苦的調伏，                                   |
| 13若已调伏祸 14则调伏苦恼                                    | 以痛苦的調伏有苦的調伏。」                                   |
| 15世之诸对象 16乃非是爱欲                                    | 「那些不是諸欲-凡世間中美的，                                |
| 17于贪之思念 18此人之爱欲                                    | 貪的意向是男子的欲，                                         |
| 19于世诸对象 20依旧犹留存                                    | 世間中美的只像那樣住立，                                     |
| 21于此之贤者 22乃调伏欲望                                    | 而在這裡明智者們調伏意欲。                                   |
| 23离忿及舍慢                                                 | 應該捨棄憤怒應該放棄慢，                                     |
| 24超越一切缚                                                 | 應該超越一切結，                                             |
| 25不执着名色                                                 | 他在名色上不執著，                                           |
| 27亦无陷入苦 26斯人无何物                                    | 苦不降臨無所有者。                                           |
| 28除念不行慢                                                 | 他捨斷名稱不來到勝慢，                                       |
| 29于此断名色 30乃至于渴爱                                    | 這裡他在名色上切斷渴愛，                                     |
| 31断结离烦恼 32乃至无欲心                                    | 那位繫結已被切斷、無苦惱、無願望者，                         |
| 36无须求其迹                                                 | 當遍求時他們沒得到：                                         |
| 33斯人此世后                                                 | 諸天、人們在此世或在他世，                                   |
| 34于人与天界 35及一切住处                                    | 在天界或在一切住處。」                                       |
| (尊者摩加拉奢如是问： )                                      | (像這樣尊者空虛王：)                                         |
| 3不见解脱人                                                  | 「如果他們沒看見那位像這樣的解脫者：                         |
| 2若天与人共 1此世及后世                                      | 諸天、人們在此世或在他世，                                   |
| 5拜无上利行                                                  | 最上的人、對人們利益的行者，                                 |
| 4以敬仰礼拜 6亦赞彼等否                                      | 凡禮敬他-他們應該被讚賞。」                                  |
| (世尊呼摩加拉奢尊者，而予回答：)                             | (世尊：「空虛王！」)                                         |
| 1比丘若礼拜 3亦应赞彼等                                      | 「那些比丘也是應該被讚賞者，                                 |
| 2如是解脱人                                                  | 凡禮敬著像這樣的那位解脫者：                                 |
| 4若知法离疑                                                  | 了知法後捨斷疑後，                                           |
| 5比丘彼等亦 6超越结缚人                                      | 那些比丘也是已超越執著者。」                                 |
