| 第七 流     | 27. 溪流經         |
| 天神：      |                    |
| 1流何处不流 | 「溪流從哪裡折回？ |
| 2涡于何处止 | 輪轉在何處不轉？   |
| 3名色何处灭 | 名與色在何處，     |
| 4无有余残留 | 被破壞無餘？」     |
| 世尊：      |                    |
| 1地水与火风 | 「於該處水與地，   |
| 2四大之灭处 | 火、風不堅立，     |
| 3由此流不流 | 從那裡溪流折回，   |
| 4于此止涡流 | 在這裡輪轉不轉，   |
| 5于此灭名色 | 在這裡名與色，     |
| 6无有余残留 | 被破壞無餘。」     |
