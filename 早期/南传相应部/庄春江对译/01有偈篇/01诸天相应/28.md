| 第八 大富                              | 28. 大富者經                            |
| 天神：                                 |                                         |
| 1有大富大财                            | 「大富者與大財富者們，                  |
| 2领国刹帝利                            | 即使擁有國家的剎帝利，                  |
| 4互为贪嫉妒                            | 他們相互貪求，                          |
| 3勿饱于爱欲                            | 是在諸欲上作不滿足者。                  |
| 5此等$$\color{red}嫉妒性$$             | 在那些生起$$\color{red}貪欲$$者中，     |
| 6随有流中流                            | 在有之流隨行者中，                      |
| 8以舍贪渴爱                            | 這裡誰捨斷了渴愛？                      |
| 7谁于世无嫉                            | 誰是世間中無貪欲者？」                  |
| 世尊：                                 |                                         |
| 1舍家以出家                            | 「捨棄家後出家者，                      |
| 2财子可爱舍                            | 捨棄兒子、家畜、所愛的後，              |
| 3以舍贪与嗔                            | 捨棄貪與瞋後，                          |
| 4以及离无明                            | 脫離無明後，                            |
| 5漏尽阿罗汉                            | 漏已滅盡的阿羅漢們，                    |
