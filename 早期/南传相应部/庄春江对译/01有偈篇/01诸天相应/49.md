| 世尊：                   |                            |
| 1此世有悭贪              | 「凡在這個世間中慳吝者們， |
| 2吝啬不与施 3詈退求乞者  | 吝嗇者們、惡口者們，       |
| 4虽是他人与              | 對其他施與者，             |
| 5亦为阻碍者              | 作障礙的人們。             |
| 6此人人乃生              | 他們會往生，               |
| 7于地狱畜生 8夜摩之世界  | 地獄、畜生界、閻摩世界，   |
| 9虽即来人界              | 如果來人的狀態，           |
| 10当生贫穷家             | 會被出生在貧困家，         |
| 12衣食游享乐             | 布料、食物、喜樂、娛樂，   |
| 11于此难获得             | 在那裡被困難地得到。       |
| 13彼等愚人望             | 愚者們從其他人希求，       |
| 14难获浮世乐             | 那也不被他們得到，         |
| 15此世有此果             | 這是在當生中的果報，       |
| 16未来堕恶道             | 且在來世為惡趣。」         |
| 天神： |      |
| 1我等知是说 | 「像這樣我們確實了知這個， |
| 2更另问瞿昙 | 喬達摩！我們問另一個， |
| 3此世得人身 | 凡在這裡人的狀態得到者們， |
| 4慈深离悭贪 | 寬容者們、離慳吝者們， |
| 5欢喜信世尊 | 在佛與法上淨信者們， |
| 6深敬法与僧 | 在僧團上極尊重者們。 |
| 8结果有如何 | 他們的果報是像什麼樣的？ |
| 7此人此世后 | 以及來世是像什麼樣的？ |
| 9今来问世尊 | 我們來到後問世尊， |

| 3欢喜信世尊 | 在佛與法上淨信者們， |

| 6光辉偏天界 | 這些輝耀天界： |
