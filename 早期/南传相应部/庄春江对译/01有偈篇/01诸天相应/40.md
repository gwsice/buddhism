| 第十 云天公主(其二)                                     | 40. 雨神的女兒經第二                                         |
| 如是我闻，                                              | 被我這麼聽聞：                                               |
| 尔时，世尊住毗舍离之大林重阁讲堂。                      | 有一次，世尊住在毘舍離大林重閣講堂。                         |
| 时，云天公主 小红莲，于黎明时其胜光偏照大林。诣世尊处， | 那時，在夜已深時，容色絕佳的雨神女兒小紅蓮使整個大林發光後，去見世尊。 |
| 诣已，礼敬世尊，立于一面。                              | 抵達後，向世尊問訊後，在一旁站立。                           |
| 立于一面之云天公主小红莲，于世尊前，唱此偈曰：          | 在一旁站立的那位雨神的女兒小紅蓮女神在世尊的面前說這些偈頌： |
| 1色鲜之电光                                             | 「這裡以閃電光輝美貌來，                                     |
| 2我云天公主 3红莲来至此                                 | 是雨神的女兒紅蓮，                                           |
| 4礼敬佛与法                                             | 禮敬佛與法地，                                               |
| 5奉说有益偈                                             | 我說這些有義理的偈頌。                                       |
| 6如此之圣法                                             | 像那樣的法，                                                 |
| 7种种之方法                                             | 我能以許多法門解析它，                                       |
| 8虽以如何多 9于分别而说 11略说此意义                    | 我將談談簡要的義理，                                         |
| 10只要我有心                                            | 以我意(心)所學得之所及。                                     |
| 13而在于此世                                            | 在一切世間中，                                               |
| 14不作一切恶 12即使身口意                               | 任何惡不應該被身語意作，                                     |
| 16于正心正念 15以离于$$\color{red}爱欲$$                | 有念正知地捨斷$$\color{red}諸欲$$後，                        |
| 18之一切苦恼 17不受无利益                               | 不應該實行苦的、伴隨無利益的。」                             |
