| 第六 天女    | 46. 天女經                |
| 天神：       |                           |
| 1天女所侍者  | 「響亮的天女眾，          |
| 2如事吸血鬼  | 是跟隨的惡鬼眾，          |
| 3此名愚痴林  | 那個樹林名為誘惑，        |
| 4如何得出口  | 怎樣將有行去[離開-㊟]？」 |
| [世尊:]      |                           |
| 1此道称真直  | 「那個道路名為正直，      |
| 2此地称无怖  | 那個方向名為無畏，        |
| 3车称无轹声  | 馬車名為不嘎嘎響，        |
| 4具足斯法轮  | 被法輪連結的。            |
| 5惭为依靠台  | 慚是它的控制工具，        |
| 6正念是帷帐  | 念是它的帷幕，            |
| 7以法我称御  | 我說法是駕駛者，          |
| 8正见是先行  | 正見是先行者。            |
| 9如是乘车人  | 凡有像這樣的車輛：        |
| 10是女或是男 | 女子的或男子的，          |
| 11依于此车乘 | 他確實以這輛車，          |
| 12到达于涅盘 | 涅槃就在他的面前。」      |
