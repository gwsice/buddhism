| 第八 只园                 | 48. 祇樹林經                 |
| 1此乃彼只园               | 「這裡確實是那個祇樹林，     |
| 2圣僧伽住处               | 仙人僧團經常來往的，         |
| 3法王请住此               | 被法王居住，                 |
| 4思之我生喜               | 我有喜的產生。               |
| 6法定及正命               | 行為、明與法，               |
| 5善意智明戒               | 戒、最上的活命，             |
| 7依此人清净               | 不免一死的人們以這些變純淨， |
| 8又依氏姓富 9非得于清净   | 非以種姓或以財產。           |
| 10是故贤智者              | 因此確實賢智的人，           |
| 11以思自己利              | 自己利益的看見者，           |
| 12并思择正法              | 應該如理檢擇法，             |
| 13如是其清净              | 這樣在那裡變成清淨。         |
| 16犹如舍利弗 14依于戒智慧 | 如舍利弗以慧，               |
| 15乃至于寂静              | 以戒以寂靜，                 |
| 17比丘到彼岸              | 凡即使到彼岸的比丘，         |
| 18应当是最胜              | 最高者會是這樣程度的。」     |
