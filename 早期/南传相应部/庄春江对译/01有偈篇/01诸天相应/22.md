| 第二 触                 | 22. 接觸經                 |
| 天神：                  |                            |
| 1不触者无触             | 「不接觸沒接觸者，         |
| 2触故应有触             | 但之後會接觸接觸者，       |
| 4于触故而触             | 因此接觸接觸者：           |
| 3故无污恶者             | [對]無犯錯者的有過失者。」 |
| 世尊：                  |                            |
| 2以污无垢者             | 「凡對無過誤的人發怒：     |
| 1清净无污恶             | 對純淨的、無污穢的人，     |
| 3其恶返愚人             | 惡就回到那位愚者，         |
| 4如同于逆风 5撒布微细尘 | 如逆風拋出的細塵。」       |
