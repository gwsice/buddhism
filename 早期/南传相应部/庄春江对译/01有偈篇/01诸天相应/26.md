| 第六 光明   | 26. 燈火經               |
| 天神：      |                          |
| 1世有几何光 | 「世間中有多少燈火，     |
| 2普照此世间 | 以該者們世間變明亮？     |
| 3我等问世尊 | 我們來問世尊，           |
| 4如何可得知 | 我們應該如何知道它？」   |
| 世尊：      |                          |
| 1世有四种光 | 「世間中有四種燈火，     |
| 2此无第五光 | 第五種在這裡沒被發現，   |
| 3于昼太阳光 | 太陽在白天照亮，         |
| 4于夜月普照 | 月亮在晚上發亮，         |
| 5时有日夜火 | 而火在白天與晚上，       |
| 6彼此皆遍照 | 到處變明亮，             |
| 7正觉最胜火 | 正覺者是照亮者中最勝的， |
| 8此为无上光 | 那是無上的光明。」       |
