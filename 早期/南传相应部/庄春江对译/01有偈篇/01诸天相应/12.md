| 第二 欢喜                              | 12.欢喜经                                |
|                                        | 起源於舍衛城。                           |
| 立于一面之天神，于世尊面前，而唱偈曰： | 在一旁站立的那位天神在世尊面前說這偈頌： |
| 1有子依子喜                            | 「有兒子者以兒子歡喜，                   |
| 2牛主依牛喜                            | 有牛者就像這樣以牛歡喜，                 |
| 3人喜依四依                            | 因為諸依著是人的歡喜，                   |
| 4无依即无喜                            | 因為凡無依著者他不歡喜。」               |
| [世尊：]                               |                                          |
| 1有子依子悲                            | 「有兒子者以兒子憂愁，                   |
| 2牛主依牛悲                            | 有牛者就像這樣以牛憂愁，                 |
| 3人悲依于依                            | 因為諸依著是人的憂愁，                   |
| 4无依即无悲                            | 因為凡無依著者他不憂愁。」               |
