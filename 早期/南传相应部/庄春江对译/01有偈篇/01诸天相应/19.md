| 元亨寺      | 庄春江                 |
| ----------- | ---------------------- |
| 第九 茅屋   | 19. 小屋经             |
| 天神：      |                        |
| 1卿有茅屋否 | 「你沒有小屋嗎？       |
| 2如何无有巢 | 沒有巢嗎？             |
| 3如何无继系 | 沒有子孫嗎？           |
| 4如何脱系缚 | 你是解脫繫縛者嗎？」   |
| 世尊：      |                        |
| 1我实无茅屋 | 「我確實沒有小屋，     |
| 2于我实无巢 | 確實沒有巢，           |
| 3我实无继系 | 確實沒有子孫，         |
| 4我实脱系缚 | 我確實是解脫繫縛者。」 |
| 天神：      |                        |
| 1何谓汝茅屋 | 「我說小屋究竟是什麼？ |
| 2何谓汝之巢 | 我說你的巢是什麼？     |
| 3何谓汝继系 | 我說你的子孫是什麼？   |
| 4何知汝系缚 | 我說繫縛是什麼？」     |
| 世尊：      |                        |
| 1汝以母谓屋 | 「你說小屋是母親，     |
| 2以妻谓巢窟 | 你說巢是妻子，         |
| 3言子为继系 | 你說子孫是兒子，       |
| 4我云渴爱缚 | 你說我的繫縛是渴愛。」 |
| 天神：      |                        |
| 1善哉卿无屋 | 「你沒有小屋-好！      |
| 2善哉无巢窟 | 沒有巢-好！            |
| 3善哉无继系 | 沒有子孫-好！          |
| 4善哉卿脱缚 | 你是解脫繫縛者-好！」  |
