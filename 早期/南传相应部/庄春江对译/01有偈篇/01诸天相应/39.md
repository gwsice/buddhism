| 第九 云天公主(其一)                                    | 39. 雨神的女兒經第一                                         |
| 如是我闻，                                             | 被我這麼聽聞：                                               |
| 尔时，世尊住毗舍离之大重阁讲堂。                       | 有一次，世尊住在毘舍離大林重閣講堂。                         |
| 时，云天之红莲公主，其胜光偏照大林。于黎明往诣世尊处， | 那時，在夜已深時，容色絕佳的雨神女兒紅蓮使整個大林發光後，去見世尊。 |
| 诣已，礼敬世尊，立于一面。                             | 抵達後，向世尊問訊後，在一旁站立。                           |
| 立于一面彼云天之红莲公主，于世尊前，唱此偈曰：         | 在一旁站立的那位雨神的女兒紅蓮女神在世尊的面前說這些偈頌：   |
| 1住毗舍离林                                            | 「在毘舍離林中居住者：                                       |
| 2众生之上首 3无上正觉者                                | 眾生中最高的正覺者，                                         |
| 5红莲恭礼奉                                            | 我是紅蓮我禮拜：                                             |
| 4我云天公主                                            | 雨神的女兒紅蓮。                                             |
| 7此法我先闻                                            | 在以前只被我聽聞：                                           |
| 6有眼者之证                                            | 被有眼者領悟的法，                                           |
| 8我今眼当知                                            | 現在我是證人我知道它：                                       |
| 9善逝牟尼说                                            | 當牟尼、善逝教導時。                                         |
| 10不论任何人 11以谤此圣法                              | 凡任何對聖法，                                               |
| 12闲荡之愚人                                           | 行斥責的劣智慧者們，                                         |
| 13堕叫唤地狱                                           | 他們到達恐怖的叫喚[地獄]，                                   |
| 14长劫受痛苦                                           | 經驗長久的苦。                                               |
| 17近此圣法人                                           | 但凡在聖法上，                                               |
| 16寂静与忍辱 15不论任何者                              | 以接受以寂靜到達者，                                         |
| 18以舍此身时                                           | 捨棄人身後，                                                 |
| 19应满天之集                                           | 將使天身充滿。」                                             |
