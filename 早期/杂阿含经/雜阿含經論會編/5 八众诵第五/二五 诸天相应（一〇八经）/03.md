# 二六；一三四九（一〇二〇）

<sup><font color="green">《相應部》（一）[〈諸天相應〉六一經](https://github.com/gwsice/buddhism/blob/master/%E6%97%A9%E6%9C%9F/%E5%8D%97%E4%BC%A0%E7%9B%B8%E5%BA%94%E9%83%A8/01%E6%9C%89%E5%81%88%E7%AF%87/01%20%E8%AF%B8%E5%A4%A9%E7%9B%B8%E5%BA%94.md#1_61)。[《別譯》二四七經](https://github.com/gwsice/buddhism/blob/master/%E6%97%A9%E6%9C%9F/%E6%9D%82%E9%98%BF%E5%90%AB%E7%BB%8F/%E5%88%AB%E8%AF%91%E6%9D%82%E9%98%BF%E5%90%AB%E7%BB%8F/12.md#247)。</font></sup>

如是我聞：一時，佛在舍衛國祇樹給孤獨園。時有天子，容色絕妙，於後夜時，來詣佛所，稽首佛足。身諸光明，遍照祇樹給孤獨園。時彼天子說偈問佛：

&emsp;&emsp;「何法映世間？&emsp;&emsp;何法無有上？

&emsp;&emsp;何等為一法，&emsp;&emsp;普制御眾生」？

爾時，世尊說偈答言：

&emsp;&emsp;「名者映世間，&emsp;&emsp;名者世無上，

&emsp;&emsp;唯有一名法，&emsp;&emsp;能制御世間」。

時彼天子復說偈言：

&emsp;&emsp;「久見婆羅門，&emsp;&emsp;逮得般涅槃，

&emsp;&emsp;一切怖已過，&emsp;&emsp;永超世恩愛」。

時彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。

# 三六；一三五九（五八三）

<sup><font color="green">《相應部》（二）[〈天子相應〉九經](https://github.com/gwsice/buddhism/blob/master/%E6%97%A9%E6%9C%9F/%E5%8D%97%E4%BC%A0%E7%9B%B8%E5%BA%94%E9%83%A8/01%E6%9C%89%E5%81%88%E7%AF%87/02%20%E5%A4%A9%E5%AD%90%E7%9B%B8%E5%BA%94.md#2_9)。[《別譯》一六七經](https://github.com/gwsice/buddhism/blob/master/%E6%97%A9%E6%9C%9F/%E6%9D%82%E9%98%BF%E5%90%AB%E7%BB%8F/%E5%88%AB%E8%AF%91%E6%9D%82%E9%98%BF%E5%90%AB%E7%BB%8F/09.md#167)。</font></sup>

如是我聞：一時，佛住舍衛國祇樹給孤獨園。爾時，羅睺羅阿修羅王，障月天子。時諸月天子，悉皆恐怖，來詣佛所，稽首佛足，退住一面。說偈歎佛：

&emsp;&emsp;「今禮最勝覺，&emsp;&emsp;能脫一切障。

&emsp;&emsp;我今遭苦惱，&emsp;&emsp;是故來歸依。

&emsp;&emsp;我等月天子，&emsp;&emsp;歸依於善逝，

&emsp;&emsp;佛哀愍世間，&emsp;&emsp;願解阿修羅」！

爾時，世尊說偈答言：

&emsp;&emsp;「破壞諸闇冥，&emsp;&emsp;光明照虛空，

&emsp;&emsp;今毘盧遮那，&emsp;&emsp;清淨光明顯。

&emsp;&emsp;羅睺避虛空，&emsp;&emsp;速放飛兔像」。

&emsp;&emsp;羅睺阿修羅，&emsp;&emsp;即捨月而還，

&emsp;&emsp;舉體悉流汗，&emsp;&emsp;戰怖不自安，

<sup><font color="green">「汗」，原本作「污」，下同。</font></sup>

&emsp;&emsp;神昏志迷亂，&emsp;&emsp;猶如重病人。

時有阿修羅名曰婆稚，見羅睺羅阿修羅疾捨月還，便說偈言：

&emsp;&emsp;「羅睺阿修羅，&emsp;&emsp;捨月一何速？

&emsp;&emsp;神體悉流汗，&emsp;&emsp;猶如重病人」？

羅睺羅<sup><font color="green">「羅」，原本缺，依宋本補。</font></sup>阿修羅說偈答言：

&emsp;&emsp;「瞿曇說呪偈，&emsp;&emsp;不速捨月者，

&emsp;&emsp;或頭破七分，&emsp;&emsp;受諸隣死苦」。

婆稚阿修羅復說偈言：

&emsp;&emsp;「佛興未曾有，&emsp;&emsp;安隱於世間，

&emsp;&emsp;說呪偈能令，&emsp;&emsp;羅睺羅捨月」。

佛說此經已，時月天子聞佛所說，歡喜隨喜，作禮而去。

# 三七；一三六〇（五八四）

<sup><font color="green">《相應部》（一）[〈諸天相應〉一九經](https://github.com/gwsice/buddhism/blob/master/%E6%97%A9%E6%9C%9F/%E5%8D%97%E4%BC%A0%E7%9B%B8%E5%BA%94%E9%83%A8/01%E6%9C%89%E5%81%88%E7%AF%87/01%20%E8%AF%B8%E5%A4%A9%E7%9B%B8%E5%BA%94.md#1_19)。[《別譯》一六八經](https://github.com/gwsice/buddhism/blob/master/%E6%97%A9%E6%9C%9F/%E6%9D%82%E9%98%BF%E5%90%AB%E7%BB%8F/%E5%88%AB%E8%AF%91%E6%9D%82%E9%98%BF%E5%90%AB%E7%BB%8F/09.md#168)。</font></sup>

如是我聞：一時，佛住舍衛國祇樹給孤獨園。時有一天子，容色絕妙，於後夜時，來詣佛所，稽首佛足，退坐一面。身諸光明，遍照祇樹給孤獨園。時彼天子說偈問佛：

&emsp;&emsp;「為有族本不？&emsp;&emsp;有轉生族耶？

&emsp;&emsp;有俱相屬無？&emsp;&emsp;云何解於縛」？

爾時，世尊說偈答言：

&emsp;&emsp;「我無有族本，&emsp;&emsp;亦無轉生族，

&emsp;&emsp;俱相屬永斷，&emsp;&emsp;解脫一切縛」。

時彼天子復說偈言：

&emsp;&emsp;「何名為族本？&emsp;&emsp;云何轉生族？

&emsp;&emsp;云何俱相屬？&emsp;&emsp;何名為堅縛」？

<sup><font color="green">「屬」，原本作「續」，依宋本改。</font></sup>

爾時，世尊說偈答言：

&emsp;&emsp;「母為世族本，&emsp;&emsp;妻名轉生族，

&emsp;&emsp;子俱是相屬，&emsp;&emsp;愛欲為堅縛。

&emsp;&emsp;我無此族本，&emsp;&emsp;亦無轉生族，

&emsp;&emsp;俱相屬亦無，&emsp;&emsp;是名脫堅縛」。

時彼天子復說偈言：

&emsp;&emsp;「善哉無族本，&emsp;&emsp;無生族亦善，

&emsp;&emsp;善哉無相屬，&emsp;&emsp;善哉縛解脫！

&emsp;&emsp;久見婆羅門，&emsp;&emsp;逮得般涅槃，

&emsp;&emsp;一切怖已過，&emsp;&emsp;永超世恩愛」。

<sup><font color="green">「怖已」，原本作「怨悉」，依宋本改。</font></sup>

時彼天子聞佛所說，歡喜隨喜，稽首佛足，即沒不現。
