## 第十一 信解品

<a name="11_1"></a>

已生与未生，能所伴受自，

谬正现非现，音生求见夺，

<a name="11_2"></a>

杂不杂违品，下广覆无覆，

勤非积未积，坚稳远行信。

<a name="11_3"></a>

多时不作意，懈怠瑜伽乱，

恶友善法微，非理之作意，

<a name="11_4"></a>

放逸及少闻，满足少闻思，

唯寂而我慢，如是许未熟，

<a name="11_5"></a>

不厌及厌离，有覆不精勤，

以及未积资，当知信解障。

<a name="11_6"></a>

大福及无悔，意乐与大乐，

不唐捐坚固，如是胜进行，

<a name="11_7"></a>

现证一切法，得自他利最，

速通彼等即，信解之功德。

<a name="11_8"></a>

欲者彼如犬，定者彼如龟，

利自彼如仆，利他彼如王。

<a name="11_9"></a>

如犬常饥苦无餍，如龟水中蜷缩住，

如仆常惧身行游，如王以令统领域，

<a name="11_10"></a>

当知散住利自他，种种信解恒超胜，

如理明知大乘胜，坚稳恒常寻求此。

<a name="11_11"></a>

因已转为人，无量每刹那，

获正等菩提，故莫甘怯懦。

<a name="11_12"></a>

犹如施他食，福德得增长，

自食非如是，利他所依法，

<a name="11_13"></a>

传讲则获得，经说大福德，

自利所依法，宣说非获得。

<a name="11_14"></a>

如是大圣广大法，具慧恒生大信解，

广大福德不断增，无量功德大本性。

## 第十二 求法品

<a name="12_1"></a>

三藏二亦可，摄故因许九。

种习了悟寂，彻证令解脱。

<a name="12_2"></a>

经对法及律，略摄许四义，

具慧了知彼，获得一切智。

<a name="12_3"></a>

说处法相法，及义故是经。

对向故反复，胜证故对法。

<a name="12_4"></a>

堕罪犯起堕，离堕数取趣，

制戒及类别，决定故为律。

<a name="12_5"></a>

承许所缘法，内外及二俱，

以二义得二，二者不可得。

<a name="12_6"></a>

依意诠说义，净信显现义，

认定唯言诠，心名安住故。

<a name="12_7"></a>

以闻等三慧，获得法所缘，

获得三所缘，真实依前宣。

<a name="12_8"></a>

三种姓行事，有无害所依，

安住于信解，别者强欲乐。

<a name="12_9"></a>

下圆二所依，有诠无诠二，

是与智相应，瑜伽因本性。

<a name="12_10"></a>

彼是间杂缘，不共而缘者，

五种及七种，此之遍知五。

<a name="12_11"></a>

具足修行相，四种三十七。

彼是二道性，二功利得受。

<a name="12_12"></a>

加行及自在，小大之本性，

瑜伽此作意，许是一切性。

<a name="12_13"></a>

恒离二迷所依者，非诠离戏真如性，

所知断净性无垢，如空金水许惑净。

<a name="12_14"></a>

众生清净别少无，无余众生愚昧心，

<sup>众生清净别少无：此句在安慧论师的《注释》中无有。</sup>

舍弃本有耽著无，世间愚相何其重！

<a name="12_15"></a>

以如幻术说，非真之妄念，

以如幻相说，迷乱为二法。

<a name="12_16"></a>

犹如彼无彼，如是许胜义，

犹如可得彼，如是许世俗。

<a name="12_17"></a>

如彼无有时，彼因了可见，

如是转依时，可见非真念。

<a name="12_18"></a>

世间未迷者，随意享彼因，

未迷持戒者，随意享转依。

<a name="12_19"></a>

彼有彼行相，非有彼实体。

如是幻术等，说为有与无，

彼有非无有，无有非是有。

<a name="12_20"></a>

于幻等亦说，有无无差别，

如是有二现，非有彼实法。

<a name="12_21"></a>

如是于色等，说为有与无，

彼有非无有，无有非是有。

于色等亦说，有无无差别。

<a name="12_22"></a>

增益与损减，诸边所破故，

为遮入下乘，如是而承许。

<a name="12_23"></a>

迷乱因迷乱，承许色之识，

及非有色识，无故另亦无。

<a name="12_24"></a>

执取幻象色，迷乱故说二，

如实虽无二，可得二者法。

<a name="12_25"></a>

执著骨锁相，迷乱故说二，

如实虽无二，可得二者法。

<a name="12_26"></a>

迷乱法相法，如是有与无，

有无无别故，有无如幻术。

<a name="12_27"></a>

凡属对治法，如是无与有，

无故无法相，亦说如幻术。

<a name="12_28"></a>

犹如魔术王，为余幻王败，

见法之佛子，彼等无我慢。

<a name="12_29"></a>

如是当了知，佛陀觉者说，

一切有为法，如幻亦如梦，

如阳焰影像，如光影回响，

犹如水月影，亦犹如幻化，

承许六六二，二六三一一。

<sup>此两颂，在藏文中为一长颂，每句十五字。</sup>

<a name="12_30"></a>

邪分别非正，非邪不分别，

非分不分别，名为诸所知。

<a name="12_31"></a>

自界显现二，无明烦恼俱，

尽生转分别，断除二实体。

<a name="12_32"></a>

获得所缘法，具住自界故，

真如无二现，趋入如皮箭。

<a name="12_33"></a>

心者显现二，承许现贪等，

或现信心等，非有惑善法。

<a name="12_34"></a>

是心现种种，趋入种种相，

显现彼有无，是故非法性。

<a name="12_35"></a>

佛为利有情，真实说事相，

法相及名相，一切细分类。

<a name="12_36"></a>

心及有见者，彼位不迁变，

此摄是事相，详分无有量。

<a name="12_37"></a>

依诠作义想，因与彼习气，

从中亦现义，遍计之法相。

<a name="12_38"></a>

依于名与义，显现名与义，

非正分别因，遍计之法相。

<a name="12_39"></a>

显现三三相，所取能取相，

非真分别念，依他起法相。

<a name="12_40"></a>

无有及有者，有无平等性，

非寂寂无别，圆成实法相。

<a name="12_41"></a>

缘于等流法，如理而作意，

安住心法界，见义有与无。

<a name="12_42"></a>

彼中成等性，圣种极无垢，

平等复超胜，无减无增相。

<a name="12_43"></a>

种子转依故，显现处境身，

转依无漏界，彼所依遍行。

<a name="12_44"></a>

意能取分别，转依无分别，

刹土智事业，是为四自在。

<a name="12_45"></a>

不动等三地，获得四自在，

一者二自在，余地许各一。

<a name="12_46"></a>

具慧证诸有，亦知二无我，

彼知平等已，持彼入真如。

复意住彼故，彼彼亦不现，

彼者乃解脱，殊胜离所缘。

<sup>“具慧证诸有……殊胜离所缘”：此两颂，在藏文中为一长颂，每颂十五字。</sup>

<a name="12_47"></a>

资粮基有生，得见唯名称，

见唯名见彼，后亦不见彼。

<a name="12_48"></a>

心即彼堕罪，我见之索缚，

极其而流转，内住许还灭。

<a name="12_49"></a>

自与自性无，自本体不住，

如执无有故，承许无本性。

<a name="12_50"></a>

前乃后所依，无性无生灭，

本来即寂灭，自性涅槃成。

<a name="12_51"></a>

始真实他性，自相自变异，

染污及差别，名无生法忍。

<a name="12_52"></a>

法无我解脱，等同种姓异，

得二意乐化，究竟故一乘。

<a name="12_53"></a>

为引有些者，为余真受持，

于诸不定者，圆佛说一乘。

<a name="12_54"></a>

声闻不定二，见未见乘义，

见义离未离，贪者此低下。

<a name="12_55"></a>

彼二得圣道，转世驾驭故，

真具不可思，驾驭之投生。

<a name="12_56"></a>

一者以愿力，而能成投生，

另者不来故，幻化尽成办。

<a name="12_57"></a>

彼二屡屡数，具有自心生，

欢喜涅槃故，许是缓证悟。

<a name="12_58"></a>

未行义投生，无有佛陀时，

求化勤禅修，获得妙菩提。

<a name="12_59"></a>

若未勤五明，大圣不成佛，

故为折摄他，自遍知精进。

<a name="12_60"></a>

得因而欢喜，所依随念彼，

欲求共同果，信解依菩提。

<a name="12_61"></a>

是以四威力，喜悦定不厌，

违品及对治，修行有四种。

<a name="12_62"></a>

净信真得受，乐于布施他，

披甲及发愿，爱乐所为意，

<a name="12_63"></a>

布施等六种，得力与成熟，

供养及依止，真实胜踊慈。

<a name="12_64"></a>

未行造恶者，惭悔欢喜境，

厌倦作敌想，安置及诠慧。

<a name="12_65"></a>

施等圆觉依，非自在天等，

以无碍解知，二者过与功，

随念积累喜，及与见大义。

<a name="12_66"></a>

瑜伽无分别，受持欲值遇，

断七颠倒执，获得之威力。

<a name="12_67"></a>

稀有无稀有，各为四种想，

于有情平等，现见大本性。

<a name="12_68"></a>

求他功德报，愿三及相续，

为修佛说故，菩萨不住此。

<a name="12_69"></a>

彼于衰兴众，不喜极欢喜。

于修伪与真，不信及起信，

不纳受作意，授记喜决定。

<a name="12_70"></a>

现见后世故，入心观平等，

以入殊胜法，认定自胜故。

<a name="12_71"></a>

十度因所生，此等善作意，

是为能恒常，增长菩萨界。

<a name="12_72"></a>

增胜意乐大，许坚稳求法，

有断无所断，如是得自在。

<a name="12_73"></a>

菩萨之寻求，无身有得身，

圆身多我慢，细慢无我慢。

<a name="12_74"></a>

坚稳法有色，无色相无疾，

神通之财富，无尽法为因。

<a name="12_75"></a>

有无与增损，一多自差别，

如名如义执，佛子真应断。

<a name="12_76"></a>

慧善依勇勤，如实求二法，

恒成众生师，如海满功德。

## 第十三 说法品

<a name="13_1"></a>

坚稳见命财，难得无实质，

极喜于苦众，恒发大布施。

法广普利众，亦非极难得，

尽施无尽增，何况非一次。

<sup>“坚稳见命财……何况非一次”：此两颂，在藏文中为一长颂，每句十七字。</sup>

<a name="13_2"></a>

各别自证故，佛陀未说法，

悲尊如巨蟒，说法如气息，

引众至彼法，寂灭如口门，

真实而安置，净广共无尽。

<sup>“各别自证故……净广共无尽”：此两颂，在藏文中为一长颂，每句十五字。</sup>

<a name="13_3"></a>

如是瑜伽修，彼则非无义，

如是善逝说，彼亦非无义。

<a name="13_4"></a>

唯闻若见义，修则成无义，

未闻若行修，说亦成无义。

<a name="13_5"></a>

菩萨尊说法，教证自在中，

一切口色法，虚空出法音。

<a name="13_6"></a>

广大断怀疑，信受示二真，

此名诸菩萨，圆满之说法。

<a name="13_7"></a>

菩萨尊说法，文不骄不厌，

明亮种种理，解无染遍行。

<a name="13_8"></a>

佛子句不弱，可意善说称，

如应及无染，适宜如是广。

<a name="13_9"></a>

略说及释说，随顺乘易解，

共称及如理，出离随同故，

<a name="13_10"></a>

略名菩萨尊，文句之圆满。

善逝语无边，六十支难思。

<a name="13_11"></a>

语句极合理，依辨除怀疑，

多番具能立，略说广解者，

<a name="13_12"></a>

佛陀此宣说，三轮皆清净，

此亦离八过，是所应了知。

<a name="13_13"></a>

懈怠及不解，不予说法机，

未定不断疑，离彼不解说。

<a name="13_14"></a>

厌烦与吝啬，此等许语过，

无有彼等故，佛说成无上。

<a name="13_15"></a>

诚信喜慧因，故此法为善，

具二义易受，示四德梵行。

<a name="13_16"></a>

与他不共同，圆断三界惑，

体性净垢白，具四梵行德。

<a name="13_17"></a>

令入相对治，及转变秘密，

于声闻本性，如是调过失，

讲说甚深性，秘密此四种。

<sup>“令入相对治……秘密此四种”：此一颂半，在藏文中为两颂。</sup>

<a name="13_18"></a>

平等及别义，别时之意趣，

数取趣意乐，当知四意趣。

<a name="13_19"></a>

轻佛及轻法，懈怠以少足，

我慢行贪欲，后悔不定退，

<a name="13_20"></a>

众生之障碍，对治说胜乘。

彼等断彼障，一切之过失。

<a name="13_21"></a>

是以句与义，受持二偈颂，

具慧有情尊，获得十功德。

<a name="13_22"></a>

界性尽增长，命终最极喜，

如愿而受生，忆念诸生世。

<a name="13_23"></a>

值遇诸佛陀，彼前闻胜乘，

信解具智慧，二门速证觉。

<a name="13_24"></a>

妙慧无厌慈，名扬知妙轨，

菩萨之说法，众中昭如日。

## 第十四 修法品

<a name="14_1"></a>

具慧于二者，断除邪与真，

以三遍了知，人法二无我。

<a name="14_2"></a>

知义即了悟，诸法如航船，

断以闻满足，故名遍知法。

<a name="14_3"></a>

异生之智慧，如是证悟二，

为圆彼智慧，随法尽修行。

<a name="14_4"></a>

菩萨于一地，与得彼本性，

一切皆平等，出世无上智。

<a name="14_5"></a>

真实尽见断，一切诸烦恼，

为断所知障，修行妙加行。

<a name="14_6"></a>

安住无分别，智慧俱同行，

如是诸余地，行持随同法。

<a name="14_7"></a>

具慧修行境，善得与善处，

善地善友伴，瑜伽具乐德。

<a name="14_8"></a>

菩萨具多闻，见谛善讲说，

无厌慈悯者，当知是大士。

<a name="14_9"></a>

缘善所依善，方便定生善，

加行善妙性，名为真专注。

<a name="14_10"></a>

欢喜生闲暇，无病及等持，

辨别能作因，即昔行福德。

<a name="14_11"></a>

是除法界外，无有他法故，

诸佛之密意，以贪离贪等。

<a name="14_12"></a>

是除法界外，无有他法故，

宣说染污法，具慧密意许。

<a name="14_13"></a>

是故于贪等，如理而趋入，

从彼得解脱，依彼离彼等。

<a name="14_14"></a>

为众住地狱，苦剧非害彼，

有寂视过功，入下乘念害。

<a name="14_15"></a>

具慧住地狱，不障大菩提，

余乘凉利己，极乐亦障彼。

<a name="14_16"></a>

当知无有法，可得无染净，

犹如幻术等，如是如虚空。

<a name="14_17"></a>

犹如如理画，本无现高低，

非真分别前，恒无二现二。

<a name="14_18"></a>

如水混浊清，澄清非彼生，

彼离彼垢已，如是心同理。

<a name="14_19"></a>

心性恒光明，是以客尘染，

法性心外心，非光明自性。

<a name="14_20"></a>

菩萨于众生，如母待独子，

彻骨大慈爱，恒常欲能利。

<a name="14_21"></a>

为利诸有情，生贪不成堕，

此与嗔恨众，诸时皆相违。

<a name="14_22"></a>

如鸽爱自子，怀抱自之子，

如是与嗔违，慈尊亦同彼。

<a name="14_23"></a>

因慈与嗔违，寂与害心违，

利与诳心违，令安违令畏。

<a name="14_24"></a>

如患依良药，能入于轮回，

如医于病患，能行众生利。

<a name="14_25"></a>

如调不成仆，能行于自身，

如商贾经商，能行诸所求。

<a name="14_26"></a>

亦如染衣人，能行诸善业，

如父于幼子，能行不害众。

<a name="14_27"></a>

如求火燧木，能行于恒常，

如托可信人，能行未圆定。

<a name="14_28"></a>

犹如幻术师，以慧修所知，

彼许菩萨尊，依上而修行。

<a name="14_29"></a>

恒具大精进，勤于二成熟，

渐以无别慧，修行趋无上。

## 第十五 教授随教品

<a name="15_1"></a>

无数劫定生，信解得增上，

如水入大海，善法真圆满。

<a name="15_2"></a>

如是积资粮，佛子原清净，

善知心善修，即是妙加行。

<a name="15_3"></a>

彼时法相续，佛前得寂止，

广大智慧故，获得广教授。

<a name="15_4"></a>

彼初勤戒者，辨别经等法，

以及不二义，心住经等名。

<a name="15_5"></a>

尔后彼依次，伺察词句类，

亦于各别内，如理析其义。

<a name="15_6"></a>

认定彼等义，复次总结法，

为得彼之义，复次彼发愿。

<a name="15_7"></a>

意诠持续寻，各自而思择，

作意无言诠，一味亦伺察。

<a name="15_8"></a>

法名总结此，当知寂止道，

当知胜观道，分析彼之义。

<a name="15_9"></a>

当知双运道，是为总结彼。

沉心极摄持，掉举亦可寂，

<a name="15_10"></a>

复于彼所缘，平等住等舍。

于彼恒常行，恭敬而修行。

<a name="15_11"></a>

专心于所缘，莫散彼相续，

散动速认知，复次安住彼。

<a name="15_12"></a>

具慧愈向上，心亦向内收，

复见功德故，等持心调柔。

<a name="15_13"></a>

由见散动过，止息不喜彼，

贪心不悦等，兴起当寂灭。

<a name="15_14"></a>

复尔勤戒者，心真实有行，

自然而获得，习彼成无行。

<a name="15_15"></a>

尔后彼身心，获得大轻安，

当知有作意，彼彼增上已，

<a name="15_16"></a>

以增上远行，彼住正行得。

希求神通者，净彼最堪能。

<a name="15_17"></a>

静虑得神通，真实成就彼，

为供无量佛，闻法往世界。

<sup>世界：藏文版本中是三界，后文注释中是世界。</sup>

<a name="15_18"></a>

依此无量劫，承侍无量佛，

于彼承侍故，心成最堪能。

<a name="15_19"></a>

是故彼等前，获得五功德。

彼成最无上，清净之法器，

一一刹那中，灭尽身粗重。

<a name="15_20"></a>

身心遍轻安，了悟诸法光，

不断真实相，未察尽得见。

<a name="15_21"></a>

如是为圆满，法身清净故，

具慧一切时，恒常摄持因。

<a name="15_22"></a>

尔后成如是，菩萨以入定，

除非意诠外，不见诸外境。

<a name="15_23"></a>

为增法光明，精进坚稳行，

法光增上已，安住唯心中。

<a name="15_24"></a>

复次诸现境，尽觉是自心，

彼时即断除，所取之散动。

<a name="15_25"></a>

尔后唯剩余，能取之散动，

彼时速触及，无间之等持。

<a name="15_26"></a>

如是彼末际，断能取散动，

此等依次第，当知是暖等。

<a name="15_27"></a>

尔后彼获得，出世无分别，

无垢离二取，无上之智慧。

<a name="15_28"></a>

彼此之转依，承许是一地，

彼此无量劫，成为极清净。

<a name="15_29"></a>

现量得彻证，法界等性时，

恒得诸众生，与己平等心。

<a name="15_30"></a>

众生同无我，痛苦行利益，

不求报心平，与余佛子同。

<a name="15_31"></a>

三界有为法，非真分别念，

彼以极清净，无二义智见。

<a name="15_32"></a>

见无彼等性，解脱诸见断，

是故决定说，彼时得见道。

<a name="15_33"></a>

了知无空性，如是有空性，

自性空性故，是名解空者。

<a name="15_34"></a>

无相之基者，分别真实灭，

无愿之彼者，非真分别念。

<a name="15_35"></a>

佛子随觉分，一切种种相，

恒见之彼道，承许俱同得。

<a name="15_36"></a>

慧行知有为，无我苦唯增，

断实无我见，得大见大义。

<a name="15_37"></a>

无我见见我，无苦成极苦，

如自于利己，利生不求报。

<a name="15_38"></a>

解胜心解脱，缚无尽久缚，

虽不见苦际，尽行而行持。

<a name="15_39"></a>

一世自痛苦，尚且不能忍，

况累世他苦，菩萨与之反。

<a name="15_40"></a>

佛子于众生，喜慈行无厌，

世间大稀有，平等非稀有。

<a name="15_41"></a>

复彼于余地，修道之此中，

修行二种智，是故能尽净。

<a name="15_42"></a>

彼无分别智，佛法能净治，

余如所安立，成熟众有情。

<a name="15_43"></a>

圆满二大劫，修道至究竟，

获得最后修，菩萨依灌顶，

<a name="15_44"></a>

得金刚喻定，分别不可坏，

转依臻究竟，无有诸障垢。

<a name="15_45"></a>

住彼即获得，尽成利群生，

了知一切相，无上之果位。

<a name="15_46"></a>

恒见难见佛，无等闻生信，

内心恒满足，无此更大义。

<a name="15_47"></a>

安住法门中，如来常亲传，

如由罪林中，牵头引菩提。

<a name="15_48"></a>

佛陀恒赞叹，真行己事善，

谴责颠倒行，最胜勤安住，

辨别之众生，断依善逝教，

广修违顺法，如理此宣说。

<sup>“佛陀恒赞叹……如理此宣说”：此两颂，在藏文中为一长颂，每句十七字。</sup>

<a name="15_49"></a>

彼见极清净，菩提无分别，

胜世除重暗，如大日普照。

<a name="15_50"></a>

恒积善充满，佛前得大教，

菩萨得广定，趋至德海岸。

*大乘经庄严论中第十五教授随教品终*

众多之信解，求法与说法，

以及修行法，教授与随教。
