## 第二十 功德品

<a name="20_1"></a>

施自身持戒，圆满皆能舍，

于弱能安忍，不顾身与命，

<a name="20_2"></a>

而发起精进，禅定无著味，

智慧无分别，具慧之稀有。

<a name="20_3"></a>

生如来种姓，得授记灌顶，

尽获大菩提，亦许大稀有。

<a name="20_4"></a>

离贪与慈悲，胜修平等心，

是故行彼等，亦非大稀有。

<a name="20_5"></a>

犹如具慧者，极慈诸有情，

众生于自妻，子亲非如是。

<a name="20_6"></a>

于求无偏堕，恒常不失戒，

普忍一切故，发起大精进。

<a name="20_7"></a>

恒常妙禅定，智慧无分别，

当知彼菩萨，如是心平等。

<a name="20_8"></a>

堪为作法器，安置于戒律，

于害能安忍，能利而行持，

<a name="20_9"></a>

令入此佛教，能断诸怀疑，

此许具慧者，饶益有情性。

<a name="20_10"></a>

恒时于众生，意乐平等摄，

令生圣者地，令善得增长，

<a name="20_11"></a>

尽防造罪恶，令听闻纯熟，

五事诸佛子，犹如众生母。

<a name="20_12"></a>

恒常于众生，能令起信心，

教增戒学等，修行解脱故，

<a name="20_13"></a>

祈请正等觉，断除彼等障，

五事诸佛子，犹如众生父。

<a name="20_14"></a>

于诸不应说，众前能保密，

谴责破学处，赞叹圆满者，

<a name="20_15"></a>

能施授教言，救护离恶魔，

五事诸佛子，犹如众生亲。

<a name="20_16"></a>

于诸染净法，自慧不迷乱，

世间出世间，圆满尽能授，

<a name="20_17"></a>

无厌无退转，恒常欲利乐，

五事诸佛子，犹如众生友。

<a name="20_18"></a>

成熟诸有情，恒常具精进，

真实说出离，安忍诸邪行，

<a name="20_19"></a>

能施二圆满，善巧彼方便，

五事诸佛子，犹如众生仆。

<a name="20_20"></a>

欲求令获得，无生之法忍，

宣说一切乘，成就瑜伽行，

<a name="20_21"></a>

相貌妙庄严，不求报异熟，

五事诸佛子，如众轨范师。

<a name="20_22"></a>

利生极精进，令圆满资粮，

积资速解脱，能断诸违品，

<a name="20_23"></a>

世间非世间，圆满皆令行，

五事诸佛子，如众亲教师。

<a name="20_24"></a>

不贪诸受用，戒律悉无失，

具有报恩者，复令行持故。

<a name="20_25"></a>

如是行六度，一切诸有情，

于诸菩萨众，如是报恩德。

<a name="20_26"></a>

恒求增与减，有情尽成熟，

于地趋胜进，求无上菩提。

<a name="20_27"></a>

断畏真发心，斩断诸怀疑，

修行之教言，佛子恒不失。

<a name="20_28"></a>

无求之布施，不求后世戒，

一切皆安忍，功德源精进。

<a name="20_29"></a>

禅定非无色，智慧具方便，

于六波罗蜜，坚稳真实行。

<a name="20_30"></a>

贪受用有失，我慢及享乐，

著味及分别，坚稳退失因。

<a name="20_31"></a>

当知住彼等，对治之菩萨，

遮除退失故，胜进具缘法。

<a name="20_32"></a>

显示与诈现，外貌显庄严，

如是暂时行，身语似寂静。

<a name="20_33"></a>

圆满通言辞，彼等离修行，

说为菩萨众，非净真实相。

<a name="20_34"></a>

反之上等行，彼等真实性。

<a name="20_35"></a>

具慧于诸地，修行布施等，

彼等能调伏，众生六违品。

<a name="20_36"></a>

以行者时别，具慧授记二，

菩提与授记，此外说为大，

<a name="20_37"></a>

获得无生忍，断我慢勤作，

与诸佛佛子，成为一本体。

<a name="20_38"></a>

复彼刹土名，时间与劫名，

眷属法住世，许为彼授记。

<a name="20_39"></a>

具慧一切时，圆满生不厌，

恒常能修行，等持无退失，

成利决定得，任运之法忍。

<a name="20_40"></a>

供养真受戒，慈悲修善法，

静处不放逸，闻义不厌足，

诸地坚稳者，无疑所应为。

<a name="20_41"></a>

了知欲妙过，思维诸迷乱，

甘受诸痛苦，修行诸善法。

<a name="20_42"></a>

不著禅乐味，不分别相状，

诸地坚稳者，恒常所应为。

<a name="20_43"></a>

法施清净戒，获得无生忍，

精进大乘法，竟具住悲心，

般若许具慧，波罗蜜多首。

<a name="20_44"></a>

以契经等别，安立为明处，

当知一切地，坚稳法安立。

<a name="20_45"></a>

依于七真如，安立真实谛。

安立理与乘，四种及三种。

<a name="20_46"></a>

如理而作意，正见具有果，

量辨不可思，当知为四理。

<a name="20_47"></a>

意乐及宣说，加行与资粮，

成就之差别，承许为三乘。

<a name="20_48"></a>

名事相互间，寻求为客性，

于此二假立，寻求唯此性。

<a name="20_49"></a>

一切不得故，真实四种智，

诸地坚稳者，能成一切利。

<a name="20_50"></a>

束缚之所依，受用种子因，

心心所及依，有种而束缚。

<a name="20_51"></a>

前存相状法，本性而住相，

悉灭具慧者，获得大菩提。

<a name="20_52"></a>

缘于真如智，断除能所取，

现前习气身，具慧能灭彼。

<a name="20_53"></a>

缘于真如智，修相无异体，

现前有无义，名驾驭分别。

<a name="20_54"></a>

凡夫障真性，普现非真实，

菩萨遣除彼，普现真实性。

<a name="20_55"></a>

不现实无有，显现真实有，

彼即为转依，随行名解脱。

<a name="20_56"></a>

境大一切时，他法现同类，

是能障碍故，遍知断除彼。

<a name="20_57"></a>

所熟所清净，所得堪成熟，

真实解说事，具慧之无量。

<a name="20_58"></a>

令生菩提心，令得无生忍，

低下无垢眼，一切漏法尽，

<a name="20_59"></a>

妙法久住世，知断及受用，

<sup>知断：也有颂词版本中是“知别”</sup>

了知行于彼，具慧说法果。

<a name="20_60"></a>

由从所缘大，如是修行大，

智慧精进大，善巧方便大，

<a name="20_61"></a>

真实成就大，佛陀事业大，

具足此大故，决定谓大乘。

<a name="20_62"></a>

种姓信解法，发起菩提心，

行持布施等，趋入无过患。

<a name="20_63"></a>

成熟诸有情，清净佛刹土，

不住之涅槃，大觉及示现。

<a name="20_64"></a>

一即具信解，增上意乐净，

有相及无相，无有现行行，

当知此五种，诸地之菩萨。

<a name="20_65"></a>

不贪三业净，伏嗔勤胜德，

法不动见真，喜菩提菩萨。

<a name="20_66"></a>

欲利无害见，堪受他害坚，

不放逸多闻，勤他利菩萨。

<a name="20_67"></a>

知执过不贪，不恨瑜伽通，

三相无恶见，乃住内菩萨。

<a name="20_68"></a>

怀慈知惭德，受苦不贪乐，

正念入定性，乘不乱菩萨。

<a name="20_69"></a>

除苦不令苦，忍苦不畏苦，

脱苦不计苦，承受苦菩萨。

<a name="20_70"></a>

不喜喜性法，诋法真勤法，

自在法无暗，首行法菩萨。

<a name="20_71"></a>

受用与调柔，守护与行善，

乐法不放逸，不放逸菩萨。

<a name="20_72"></a>

不敬微过惭，不忍退失惭，

流散小见惭，他乘惭菩萨。

<a name="20_73"></a>

此他与等舍，具行得自在，

相应说大果，菩萨行饶益。

<a name="20_74"></a>

菩萨摩诃萨，具慧胜光明，

佛子佛陀基，能胜如来芽，

<a name="20_75"></a>

具力及圣尊，商主与胜称，

悲尊大福德，自在及具法。

<a name="20_76"></a>

证真证大利，证诸义恒证，

以及证方便，是故名菩萨。

<a name="20_77"></a>

随我证细见，证悟种种识，

证非真分别，是故名菩萨。

<a name="20_78"></a>

证未证随证，证无实证生，

各别证未证，是故名菩萨。

<a name="20_79"></a>

证实无胜义，证诸义所有，

证所证证基，是故名菩萨。

<a name="20_80"></a>

证成就证处，证胎证次第，

证悟除怀疑，是故名菩萨。

<a name="20_81"></a>

具慧慧证得未得，善住证悟随证悟，

妙说无说有无我，未熟真实成熟慧。

## 第二十一 行为究竟品

<a name="21_1"></a>

慈悲说爱语，坚稳与伸手，

决定解密意，此等具慧相。

<a name="21_2"></a>

摄受得信解，不厌摄集故，

意乐加行中，可知五种相。

<a name="21_3"></a>

菩萨于恒常，在家成轮王，

生生世世中，利益诸有情。

<a name="21_4"></a>

诸地具慧者，出家由受得，

及以法性得，余示出家相。

<a name="21_5"></a>

诸位出家者，功德无有量。

精勤持戒者，胜在家菩萨。

<a name="21_6"></a>

诸地坚稳者，他世令悦果，

此生令行善，欲令得涅槃。

<a name="21_7"></a>

不净净极净，许众生意乐。

<a name="21_8"></a>

诸地具慧者，发愿心平等，

成为君主者，集眷普摄持。

<a name="21_9"></a>

承许具慧者，以业以愿力，

以及等持力，自在而投生。

<a name="21_10"></a>

法相人学处，蕴及成就相，

释词与获得，安住即地者。

<a name="21_11"></a>

殊胜证空性，诸业不虚住，

安住于禅乐，其后生欲界。

<a name="21_12"></a>

复次菩提分，回向于轮回，

内心无烦恼，成熟诸有情。

<a name="21_13"></a>

故意而投生，随护染污法，

与一行道系，唯定无相道。

<a name="21_14"></a>

无相任运成，清净诸刹土，

其后则成就，成熟诸众生。

<a name="21_15"></a>

等持与总持，菩提清净性，

依此等安立，当知地法相。

<a name="21_16"></a>

菩萨见清净，戒极净入定，

断法慢相续，染净异慢无。

<a name="21_17"></a>

刹那获得慧，净土善熟众，

大力身圆满，善巧示灌顶。

<a name="21_18"></a>

证悟诸法性，于此随学修，

增上戒定慧，智慧二行境，

<a name="21_19"></a>

法之真实性，入不知知等，

智慧行境故，二地住有彼。

<a name="21_20"></a>

学修及修行，余有四种果：

无相有行住，是为第一果；

<a name="21_21"></a>

真如无现行，清净诸刹土，

承许第二果；成熟诸众生；

<a name="21_22"></a>

等持及总持，成就微妙果。

此等四种果，真实依四地。

<a name="21_23"></a>

尽证法性已，于此净戒蕴，

其后则清净，定蕴及慧蕴。

<a name="21_24"></a>

此外诸余地，解脱四种障，

亦解质碍障，清净解脱智。

<a name="21_25"></a>

当知一切地，未成就成就，

成就亦承许，未成就成就。

<a name="21_26"></a>

当知成就地，作意依安立，

了知是分别，彼亦不分别。

<a name="21_27"></a>

各别自证故，佛陀行境故，

修行一切地，成就不可思。

<a name="21_28"></a>

有现及信解，不怯不软弱，

不仰仗他者，一切尽证悟。

<a name="21_29"></a>

于诸心平等，无转无贪爱，

知方便生眷，承许是众相。

<a name="21_30"></a>

并非无希求，无贪恼嗔怠，

无非慈悲意，恶慧不可夺。

无散乐不夺，不为痛苦害，

依师勤闻法，精进供养佛。

<sup>“并非无希求……精进供养佛”此两颂，藏文是一长颂，每句十七字，因过长而译为两句。</sup>

<a name="21_31"></a>

了知胜方便，真汇广大福，

共同一日中，回向圆菩提。

生善恒行善，神通功德戏，

当知彼佛子，最上功德藏。

<sup>“了知胜方便……最上功德藏”此两颂，藏文是一长颂，每句十七字，因过长而译为两句。</sup>

<a name="32"></a>

诸地一切时，具慧之功德，

承许止与观，二俱五本性。

<a name="21_33"></a>

见近于菩提，成办众生利，

生起极欢喜，故名极喜地。

<a name="21_34"></a>

离破戒勤垢，故名无垢地，

放法大光明，故名发光地。

<a name="21_35"></a>

如此菩提分，具有尽焚焰，

彼地焚二障，是故名焰慧。

<a name="21_36"></a>

成熟诸众生，亦能护自心，

具慧行难行，故名难行地。

<a name="21_37"></a>

依于般若度，复次现前此，

轮回与涅槃，故名现前地。

<a name="21_38"></a>

随系一行道，故许远行地，

不为想动摇，故名不动地。

<a name="21_39"></a>

无碍解妙慧，故名善慧地，

如云遍法空，故名法云地。

<a name="21_40"></a>

于行种种善，欢喜恒安住，

菩萨之诸地，承许名为处。

<a name="21_41"></a>

无量无畏故，不可估量故，

后后胜进故，彼等许为地。

<a name="21_42"></a>

得地有四种，即是以信解，

趋入诸行为，证悟及成就。

<a name="21_43"></a>

信解大乘者，信解小乘者，

二者为欲求，及与为调化，

坚稳之四行，依经随同说。

<a name="21_44"></a>

慈悲有情尊，值意离意尊，

不离意趣尊，利乐汝前礼。

<a name="21_45"></a>

决定脱诸障，胜伏诸世间，

智遍诸所知，心解汝前礼。

<a name="21_46"></a>

摧毁诸众生，无余诸烦恼，

破惑于惑众，悲悯汝前礼。

<a name="21_47"></a>

任运无贪执，无碍恒入定，

能答诸所问，是尊汝前礼。

<a name="21_48"></a>

所能依所说，语知为能说，

智慧恒无碍，善说汝前礼。

<a name="21_49"></a>

前往知众行，了知彼等语，

来去及出离，善授汝前礼。

<a name="21_50"></a>

有情现见您，知是大丈夫，

唯见令极喜，是尊汝前礼。

<a name="21_51"></a>

受住及舍弃，幻化及转变，

等持与智慧，自在汝前礼。

<a name="21_52"></a>

方便皈处净，于大乘出离，

欺惑众生魔，摧之汝前礼。

<a name="21_53"></a>

说自智与断，他利离道障，

一切诸外道，不损汝前礼。

<a name="21_54"></a>

无护无忘失，眷中庄严语，

永断二染污，摄眷汝前礼。

<a name="21_55"></a>

遍知您出游，安住皆恒常，

无非遍知行，如实汝前礼。

<a name="21_56"></a>

于利诸众生，您恒不越时，

恒常行有义，不忘汝前礼。

<a name="21_57"></a>

于诸世间中，昼夜六时见，

具足大悲心，垂念汝前礼。

<a name="21_58"></a>

行为与证悟，智慧及事业。

声闻缘觉众，上师汝前礼。

<a name="21_59"></a>

三身大菩提，证得一切相，

于诸众生疑，尽断汝前礼。

<a name="21_60"></a>

无执无过失，无垢不舍置，

不动于诸法，无戏汝前礼。

<a name="21_61"></a>

成就殊胜义，出离一切地，

成为众生尊，令众得解脱。

<a name="21_62"></a>

无等无尽德，显现诸世间，

亦现佛坛城，人天皆不见。

<a name="21_63"></a>

此亦依彼力，如随所化缘，

世界存在间，事业不间断。
