## 第十六 方便品

<a name="16_1"></a>

林众生山河，所依终是地，

智者三种业，是施等善依。

<a name="16_2"></a>

极难业众体，累劫竟勤尊，

佛子身语意，性业净无厌。

<a name="16_3"></a>

如欲利己身，离毒刃雳敌，

佛子三业性，能遮二下乘。

<a name="16_4"></a>

恒时不分别，作者业所作，

此真方便摄，业净至彼岸。

## 第十七 六度四摄品

<a name="17_1"></a>

数法相次第，定义修功德，

分摄违品德，互定当了知。

<a name="17_2"></a>

受用身眷行，圆满增上生，

恒不随惑转，所为不颠倒。

<a name="17_3"></a>

精勤众生利，施不害忍行，

处解脱俱基，是行诸自利。

<a name="17_4"></a>

不贫不恼害，忍害行不厌，

令喜善说故，他利即自利。

<a name="17_5"></a>

不爱诸受用，极敬不厌二，

瑜伽无分别，大乘仅此已。

<a name="17_6"></a>

于境不贪道，得彼禁散乱，

不舍众生增，余者净障碍。

<a name="17_7"></a>

是就三学言，佛真说六度，

初三末二者，二学一属三。

<a name="17_8"></a>

布施断违品，具不分别智，

满足诸所愿，成熟有情三。

<a name="17_9"></a>

持戒断违品，具不分别智，

满足诸所愿，成熟有情三。

<a name="17_10"></a>

安忍断违品，具不分别智，

满足诸所愿，成熟有情三。

<a name="17_11"></a>

精进断违品，具不分别智，

满足诸所愿，成熟有情三。

<a name="17_12"></a>

禅定断违品，具不分别智，

满足诸所愿，成熟有情三。

<a name="17_13"></a>

智慧断违品，具不分别智，

满足诸所愿，成熟有情三。

<a name="17_14"></a>

依前而生后，下劣殊胜故，

粗大细微故，如此说次第。

<a name="17_15"></a>

能令离贫穷，获清凉灭嗔，

<sup>灭嗔：本人所用《弥勒五论颂词》中是忍嗔，安慧论师的《注释》中颂词是灭嗔。</sup>

胜行及持心，了知胜义故。

<a name="17_16"></a>

是说一切修，依实依作意，

依思依方便，以及依自在。

<a name="17_17"></a>

能舍一切事，依于根本心，

圆满身受用，摄二能圆满，

<a name="17_18"></a>

彼亦具无吝，法财无畏施，

遍知施如是，智者真实行。

<a name="17_19"></a>

六支寂意摄，善趣赐安住，

所依灭无畏，具足福德资，

<a name="17_20"></a>

表示法性得，安住戒者具，

遍知戒如是，智者真实行。

<a name="17_21"></a>

堪害耐苦知，悲悯依法故，

真说五功德，彼能行二利，

<a name="17_22"></a>

具诸难行首，承许彼三种，

遍知忍如是，智者真实行。

<a name="17_23"></a>

真实喜善法，依信及欲乐，

念等功德增，烦恼之对治，

<a name="17_24"></a>

具无贪等德，彼等有七种，

遍知勤如是，智者真实行。

<a name="17_25"></a>

心于内安住，依正念精进，

能生诸安乐，驾驭神通住，

<a name="17_26"></a>

位居白法首，彼等具彼三，

遍知禅如是，智者真实行。

<a name="17_27"></a>

正辨诸所知，依于三摩地，

解脱诸染污，善说续慧命，

<a name="17_28"></a>

一切法之最，彼等具彼三，

遍知慧如是，智者真实行。

<a name="17_29"></a>

当知诸白法，散乱入定俱，

是以二二二，波罗蜜多摄。

<a name="17_30"></a>

菩萨施未贪，非贪无贪著，

亦非贪未贪，非贪无贪著。

<a name="17_31"></a>

菩萨戒未贪，非贪无贪著，

亦非贪未贪，非贪无贪著。

<a name="17_32"></a>

菩萨忍未贪，非贪无贪著，

亦非贪未贪，非贪无贪著。

<a name="17_33"></a>

菩萨勤未贪，非贪无贪著，

亦非贪未贪，非贪无贪著。

<a name="17_34"></a>

菩萨定未贪，非贪无贪著，

亦非贪未贪，非贪无贪著。

<a name="17_35"></a>

菩萨慧未贪，非贪无贪著，

亦非贪未贪，非贪无贪著。

<a name="17_36"></a>

佛子遇乞命亦舍，悲不图报所欲果，

施将众置三菩提，智摄世间存无尽。

<a name="17_37"></a>

佛子恒受禁勤戒，不求善趣得不贪，

戒将众置三菩提，智摄世间存无尽。

<a name="17_38"></a>

佛子忍极难行害，非求善弱畏见利，

忍将众置三菩提，智摄世间存无尽。

<a name="17_39"></a>

佛子铠行勤无喻，摧自他惑得妙觉，

勤将众置三菩提，智摄世间存无尽。

<a name="17_40"></a>

佛子众定成静虑，妙禅乐住悲生下，

定将众置三菩提，智摄世间存无尽。

<a name="17_41"></a>

佛子知如尽所知，不著涅槃况轮回？

智将众置三菩提，摄众世间存无尽。

<a name="17_42"></a>

广大及无染，大义与无尽，

当知是施等，一切四功德。

<a name="17_43"></a>

乞见圆满喜，不喜及慕求，

施者具悲悯，超胜恒胜伏。

<a name="17_44"></a>

慈悲于众生，性命受用妻，

恒施极欢喜，何不护断彼？

<a name="17_45"></a>

圣不惜心平，无畏悲普施，

岂能以害他，言说颠倒语？

<a name="17_46"></a>

悲欲平等利，极畏他生苦，

心勤调众生，远离语三罪。

<a name="17_47"></a>

普施具悲心，精通缘起法，

如何能甘受，意之诸烦恼？

<a name="17_48"></a>

悲尊受害时，利他于自苦，

恒得益想喜，于何何所忍？

<a name="17_49"></a>

于他离他想，恒爱他胜己，

难行故悲尊，勤非难极难。

<a name="17_50"></a>

三定小安乐，自乐执退失，

穷尽许有痴，菩萨定反之。

<a name="17_51"></a>

如夜手摸触，罩灯三种慧，

悲尊之智慧，无比如日光。

<a name="17_52"></a>

布施所依事，缘由普回向，

因智田及处，承许最殊胜。

<a name="17_53"></a>

持戒所依事，缘由普回向，

因智田及处，承许最殊胜。

<a name="17_54"></a>

安忍所依事，缘由普回向，

因智田及处，承许最殊胜。

<a name="17_55"></a>

精进所依事，缘由普回向，

因智田及处，承许最殊胜。

<a name="17_56"></a>

禅定所依事，缘由普回向，

因智田及处，承许最殊胜。

<a name="17_57"></a>

智慧所依事，缘由普回向，

因智田及处，承许最殊胜。

<a name="17_58"></a>

菩萨之布施，为一众生乐，

累劫受害喜，何况实反之。

<a name="17_59"></a>

众生为求财，坚稳施众生，

众为身求财，坚稳百番施。

<a name="17_60"></a>

施身尚心乐，轻物何须说，

彼喜出世间，是为最无上。

<a name="17_61"></a>

具慧施一切，求者喜而喜，

乞者得所乞，欢喜非如是。

<a name="17_62"></a>

具慧施无财，思自具财富，

乞者圆受用，不思自富有。

<a name="17_63"></a>

具慧施如理，满愿思大益，

乞者得多财，不感施者恩。

<a name="17_64"></a>

如享路边果，有情受广财，

无虑自受用，乐施唯菩萨。

<a name="17_65"></a>

首因作用别，相别所依别，

四违对治别，知精进佛说。

<a name="17_66"></a>

善资精进胜，依彼随得彼，

精进胜乐住，成就世出世。

<a name="17_67"></a>

精进得世愿，精进净成上，

精进离坏聚，精进证大觉。

<a name="17_68"></a>

减增之精进，解脱断违品，

入真如转依，大义许别词。

<a name="17_69"></a>

初善如理行，不怯及不退，

不满之精进，诸相佛尽宣。

<a name="17_70"></a>

依勤三乘士，意慧具弱广，

下中上精进，承许小大义。

<a name="17_71"></a>

精进财不败，精进惑不败，

精进厌不败，精进得不败。

<a name="17_72"></a>

互摄类别故，法性故因故，

六度一切时，决定当了知。

<a name="17_73"></a>

承许布施同，说彼令行持，

自随行爱语，利行及同事。

<a name="17_74"></a>

能利之方便，令持及令行，

如是能随行，当知四摄事。

<a name="17_75"></a>

初者令成器，第二令信解，

第三令行持，第四令清净。

<a name="17_76"></a>

财摄及法摄，所缘之法等，

是以二种摄，许为四摄事。

<a name="17_77"></a>

当知以相别，分下中上摄，

多无义有义，一切皆有义。

<a name="17_78"></a>

相应摄眷属，真实依此理，

普成一切利，佛赞妙方便。

<a name="17_79"></a>

已摄及将摄，现今正摄受，

如是彼一切，成熟众生道。

<a name="17_80"></a>

恒不贪受用，寂禁勤住性，

不分有境相，依彼摄群生。

## 第十八 供养依师无量心品

<a name="18_1"></a>

具足清净心，为圆二资粮，

佛前以衣等，现非现量供。

<a name="18_2"></a>

发愿佛出世，自他皆有义，

不缘彼三轮，是圆满供佛。

<a name="18_3"></a>

余者能成熟，无量诸有情，

是以物内心，信解与发愿，

<a name="18_4"></a>

悲悯及安忍，真实行持门，

专注基证悟，解脱真如别。

<a name="18_5"></a>

缘由普回向，所依事与因，

智慧以及田，处供尽宣说。

<a name="18_6"></a>

因果及自他，利养恭敬性，

修行彼分二，小大有无慢，

加行与行趣，以愿许远近。

<sup>此一颂半，藏文为两颂。</sup>

<a name="18_7"></a>

自心供养佛，信法思自在，

无别方便摄，普入一义胜。

<a name="18_8"></a>

所依依止事，缘由普回向，

因智净佛土，及处尽宣说。

<a name="18_9"></a>

依师当调柔，寂灭功德胜，

勤富教悟真，善说悲无厌。

<a name="18_10"></a>

恭敬利养侍，修行依上师，

依事具慧者，知法意敬往。

<a name="18_11"></a>

彼无求利养，为修普回向，

坚稳依教行，真令师心喜。

<a name="18_12"></a>

通三乘慧悟，精勤修自乘，

成熟无量众，修行佛净土。

<a name="18_13"></a>

为享法缘德，依止非为财。

具慧因与果，法门随行外，

<a name="18_14"></a>

闻思及瑜伽，有无慢依师。

具慧以行趣，加行愿依师。

<a name="18_15"></a>

自心依上师，信法思自在，

无别方便摄，普入一义胜。

<a name="18_16"></a>

坚稳梵断违，具无分别慧，

趋入三所缘，成熟诸有情。

<a name="18_17"></a>

坚稳缘求乐，苦逼具乐者，

具惑彼等法，悟入彼真如。

<a name="18_18"></a>

彼义真如故，得寂清净故，

二业灭惑故，乃无缘慈心。

<a name="18_19"></a>

彼等动不动，著味不著味，

菩萨不动摇，善住离贪中。

<a name="18_20"></a>

非入定本性，小中及下地，

意劣有我慢，皆下余为胜。

<a name="18_21"></a>

具慧住梵住，恒常生欲界，

圆满诸资粮，成熟众有情。

<a name="18_22"></a>

悉不离梵住，远离彼违品，

放逸彼之缘，猛烈亦不变。

<a name="18_23"></a>

菩萨具害心，损恼不欢喜，

嗔心及贪心，招致众过患。

<a name="18_24"></a>

惑毁自他戒，有失利养微，

护神师责罚，恶名转无暇，

失得未得故，心生大痛苦。

<sup>此六句藏文中是一长颂，每句十三字。</sup>

<a name="18_25"></a>

善住慈心等，不生此诸过，

非惑为利生，不舍于轮回。

<a name="18_26"></a>

佛子于有情，慈等心生此，

众生于有德，独子非如是。

<a name="18_27"></a>

炽盛随敌转，苦逼暗遮覆，

普入难行道，真具大束缚。

<a name="18_28"></a>

贪杂剧毒食，舍道入歧途，

弱势之有情，是生悲心境。

<a name="18_29"></a>

断害大觉种，生乐恼欲因，

本性依五法，佛子近菩提。

<a name="18_30"></a>

大悲胜慧知，轮回皆苦性，

无我而不厌，亦不为过损。

<a name="18_31"></a>

世间苦视己，悲尊如实知，

彼苦净方便，爱怜亦不厌。

<a name="18_32"></a>

悲尊此悲悯，自性妙抉择，

先前具串习，除违得清净。

<a name="18_33"></a>

非等恒胜意，非行非离贪，

非无缘非悲，无悲非菩提。

<a name="18_34"></a>

悲忍及思维，愿生成熟众，

初根终胜果，此是悲妙树。

<a name="18_35"></a>

若无根本悲，不能忍苦行，

具慧不耐苦，不思利众生。

<a name="18_36"></a>

离思不发愿，清净之投生，

不得善妙生，不能成熟众。

<a name="18_37"></a>

悲润是慈心，彼苦以乐广，

如理而作意，树枝繁茂增。

<a name="18_38"></a>

愿力不间断，舍受叶二缘，

成就故花果，得生当了知。

<a name="18_39"></a>

大悲功德源，谁不悯众生？

为此纵痛苦，悲生无量乐。

<a name="18_40"></a>

悲摄慈悯尊，尚不住寂灭，

何言贪世间，安乐及自命？

<a name="18_41"></a>

彼慈非无罪，亦非出世间，

具慧慈悲心，无罪出世间。

<a name="18_42"></a>

痛苦汹涛中，痴重暗世间，

引导之方便，岂不成无罪？

<a name="18_43"></a>

世菩提缘觉，罗汉无彼慈，

其余何须言，岂非出世间？

<a name="18_44"></a>

菩萨无痛苦，悲悯成痛苦，

彼初普畏怖，证得极欢喜。

<a name="18_45"></a>

悲生苦胜伏，世间一切乐，

行利尚不具，何有更稀奇？

<a name="18_46"></a>

坚稳悲布施，布施令安乐，

享用三界乐，不及彼一分。

<a name="18_47"></a>

利生以悲悯，不舍苦轮回，

悲尊为利他，何不甘受苦？

<a name="18_48"></a>

悲尊悲布施，受用彼生慈，

摄受生彼力，成乐恒增长。

<a name="18_49"></a>

以悲广行增，以施成熟乐，

一切普引领，如教懦弱者。

<a name="18_50"></a>

悲苦成痛苦，苦者未得乐，

悲尊何能乐？他乐自方乐。

<a name="18_51"></a>

悲尊教自施，不求自安乐，

受用令他乐，不异自非乐。

<a name="18_52"></a>

彼乐乐己施，及果施众生，

汝于我有作，果皆予彼等。

<a name="18_53"></a>

施者不求财，财丰遇善妙，

我非求安乐，如此辗转施。

<a name="18_54"></a>

我常以悲悯，一切皆施舍，

待彼汝当知，我不求彼果。

<a name="18_55"></a>

得果若不舍，不成乐布施，

刹那有不施，非是乐施者。

<a name="18_56"></a>

未作不生果，希报非我宗，

无视汝回报，果报尽施他。

<a name="18_57"></a>

佛子悲悯施，无罪清净物，

引乐予随护，无寻无染污。

<a name="18_58"></a>

佛子悲悯施，一切广主常，

欢喜无染净，向觉亦向善。

<a name="18_59"></a>

三乐心意足，悲尊普施喜，

享用受用者，非有如是喜。

<a name="18_60"></a>

悲贫悲恶劣，悲乱悲放逸，

悲悯随境转，悲执颠倒者。

<a name="18_61"></a>

由乐苦彼因，菩萨生悲心，

由因师自性，菩萨生悲心。

<a name="18_62"></a>

菩萨之悲心，当知是平等，

意乐行离贪，不缘清净故。

<a name="18_63"></a>

自心修慈等，信法及思维，

自在无分别，合一故最胜。

<a name="18_64"></a>

于佛生净信，物敬持续供，

依多德益师，悲众得成就。

## 第十九 菩提分品

<a name="19_1"></a>

知惭断违品，具无分别智，

无罪下劣境，熟众坚稳相。

<a name="19_2"></a>

菩萨于六度，违增对治减，

生起大惭愧，坚稳依六度，

懈怠随惑法，享用更惭愧。

<sup>此六句，藏文为两颂。</sup>

<a name="19_4"></a>

非入定本性，小中及下地，

意劣有我惭，皆下余为胜。

<a name="19_5"></a>

具慧无惭愧，非理纳烦恼，

嗔舍及我慢，摧毁众生戒。

<a name="19_6"></a>

后悔及丧失，恭敬利养微，

具信非人众，本师亦置之。

<a name="19_7"></a>

同行佛子责，于世间人中，

现法得恶名，他世生无暇。

<a name="19_8"></a>

是故失已得，未得诸白法，

由此处痛苦，心亦不正住。

<a name="19_9"></a>

有惭诸佛子，不生此诸过，

智者常真实，受生人天界。

<a name="19_10"></a>

具慧以有惭，速圆正觉资，

佛子不厌倦，成熟众有情。

<a name="19_11"></a>

彼恒离违品，不离对治法，

有惭之佛子，得彼等功德。

<a name="19_12"></a>

凡愚著妙衣，离愧有罪垢，

佛子知惭衣，无遮离罪垢。

<a name="19_13"></a>

佛子具惭愧，如空法不染，

佛子内集聚，知惭妙庄严。

<a name="19_14"></a>

菩萨以惭愧，如母慈所化，

惭愧于轮回，能防一切罪。

<a name="19_15"></a>

能受于一切，亦不受一切，

不行行一切，愧者愧成相。

<a name="19_16"></a>

自心修惭愧，信解法思维，

自在无分别，合一堪首位。

<a name="19_17"></a>

菩萨之坚稳，较彼其余众，

以法相分类，不变而超胜。

<a name="19_18"></a>

精进定与慧，许为勇耐稳，

菩萨以此三，无惧尽趋入。

<a name="19_19"></a>

怯懦动摇痴，诸事生畏惧，

故知三本性，是为坚稳名。

<a name="19_20"></a>

自性及宏愿，无所顾惜性，

众生颠倒行，听闻深广法，

<sup>听闻深广法：麦彭仁波切的此《注疏》中是信解深广法，安慧论师的《注释》中是听闻，也有其他版本中是信解。</sup>

<a name="19_21"></a>

所化难调化，佛身不可思。

种种诸难行，轮回皆不舍，

<a name="19_22"></a>

无有烦恼性，坚稳生坚稳，

其余无伦比，许具坚稳最。

<a name="19_23"></a>

飞蛾飞禽海，不能动须弥，

恶友苦闻深，不能动坚稳。

<a name="19_24"></a>

菩萨不厌最，三事闻不足，

精勤耐受苦，依于惭坚稳。

<a name="19_25"></a>

欲求大菩提，具慧之不厌，

诸地未圆满，圆满极圆满。

<a name="19_26"></a>

坚稳知论典，基所为业相，

无尽成果胜，等持总持摄，

能成熟有情，亦能持妙法。

<sup>此一颂半，藏文为两颂。</sup>

<a name="19_28"></a>

坚稳知世间，是以身语意，

知谛无伦比，超胜彼其余。

<a name="19_29"></a>

坚稳常含笑，诚实而言语，

为众堪法器，为令修妙法。

<a name="19_30"></a>

因许以二谛，令世间恒生，

以二令隐没，了此知世间。

<a name="19_31"></a>

为寂为获得，具慧修真谛，

具慧知谛故，定称知世间。

<a name="19_32"></a>

依于经典法，彼义具密意，

了义具正量，获得彼无诠。

<a name="19_33"></a>

是为遮舍法，依文倒执真，

获得有言诠，于此说四依。

<a name="19_34"></a>

于信解分析，他前如实闻，

无诠智慧中，坚稳不退失。

<a name="19_35"></a>

异门及法相，词智了知故，

菩萨无碍解，四种无伦比。

<a name="19_36"></a>

极其精进说，说何以何说，

所说法与义，唯以词智说。

<a name="19_37"></a>

法即略广说，诸时得二者，

答复辩方故，无碍解为四。

<a name="19_38"></a>

等性各自证，复次尽了知，

能断一切疑，名为无碍解。

<a name="19_39"></a>

菩萨之资粮，福慧无伦比，

福得增上生，依慧无惑转。

<a name="19_40"></a>

施戒为福资，智慧乃慧资，

余三属二资，五度皆慧资。

<a name="19_41"></a>

平时得修行，反复行持善，

坚稳之资粮，能成一切利。

<a name="19_42"></a>

是为趋入地，无相任运成，

灌顶及究竟，坚稳修资粮。

<a name="19_43"></a>

具慧修念住，是以十四相，

无等故胜余，所依对治入，

所缘与作意，获得观修胜。

<a name="19_44"></a>

随应及随行，遍知与受生，

广大殊妙性，修行真成就。

<a name="19_45"></a>

坚稳之正断，有情无伦与，

是为修念住，过失之对治。

<a name="19_46"></a>

轮回享受用，为除诸盖障，

为舍劣作意，为入一地等。

<a name="19_47"></a>

为住于无相，为获佛授记，

为成熟有情，是为得灌顶。

<a name="19_48"></a>

为净佛刹土，是为究竟性，

具慧诸菩萨，修违品对治。

<a name="19_49"></a>

依于欲乐已，瑜伽有相修，

一切诸正断，决定称对治。

<a name="19_50"></a>

坚稳四神足，法相具超胜，

是为成自他，一切利益生。

<a name="19_51"></a>

所依与分类，方便及修行，

安立具慧者，神足一切相。

<a name="19_52"></a>

依于禅定度，是说四类别，

以及四方便，六种真成就。

<a name="19_53"></a>

一是能精勤，第二饶益性，

第三能专注，第四为对治。

<a name="19_54"></a>

能见及教言，安住行游戏，

宏愿与自在，得法成就故。

<a name="19_55"></a>

菩提及行为，闻胜与止观，

当知信等基，能成利益时。

<a name="19_56"></a>

信等趋入地，许为有染污，

违品薄弱故，彼等称为力。

<a name="19_57"></a>

承许入地者，安立为觉支。

证悟一切法，众生等性故。

<a name="19_58"></a>

了知克胜故，正念即普行，

此之择法慧，能摧分别相。

<a name="19_59"></a>

此之精进支，为速证悟入，

正法光明增，欢喜恒长养。

<a name="19_60"></a>

脱离一切障，轻安获得乐，

由定自在得，心中所想事。

<a name="19_61"></a>

等舍于一切，随意而安住，

后得无分别，安住恒住胜。

<a name="19_62"></a>

此德之菩萨，安立如轮王，

常以如七宝，觉支所围绕。

<a name="19_63"></a>

自性安住支，第三出离支，

第四功德支，三种无惑支。

<a name="19_64"></a>

其后则随行，如实所证悟，

如实证安立，趋入所安立。

<a name="19_65"></a>

三业皆清净，修行诸所知，

正道胜功德，障碍之对治。

<a name="19_66"></a>

依于正安住，住心为寂止，

尽辨一切法，是故为胜观。

<a name="19_67"></a>

止观即遍行，一分非一分，

许因证出离，无相及无行。

<a name="19_68"></a>

清净极清净，止观之瑜伽，

遍行一切地，能成一切利。

<a name="19_69"></a>

圆满佛陀法，成熟诸有情，

速得成所为，精通道不断。

<a name="19_70"></a>

菩萨一切地，方便无伦比，

依彼巧方便，能成一切利。

<a name="19_71"></a>

异熟闻串习，等持之总持，

小品及大品，大品亦有三。

<a name="19_72"></a>

具慧未入小，入地不清净，

诸地为中品，净地总持大。

<a name="19_73"></a>

如是诸菩萨，反复真依彼，

常宣及受持，微妙之正法。

<a name="19_74"></a>

坚稳之宏愿，具思维欲乐，

彼依智劝勉，诸地无伦比。

<a name="19_75"></a>

当知彼为因，唯心成就故，

由心即具果，彼成未来利。

<a name="19_76"></a>

种种大菩提，诸地愈向上，

彼愿愈清净，菩萨成二利。

<a name="19_77"></a>

三等持行境，乃为二无我，

执我之所依，恒常善寂灭。

<a name="19_78"></a>

所取能取体，当知有三种，

无分别背离，具足极欢喜。

<a name="19_79"></a>

是为遍了知，断除及现前，

普宣空性等，三种等持义。

<a name="19_80"></a>

欲利诸有情，三种等持因，

四种法印义，说于诸菩萨。

<a name="19_81"></a>

坚稳四法印，无义分别义，

唯是遍计义，寂灭分别义。

<a name="19_82"></a>

不容因生故，相违自不住，

无故相定故，随行泯灭故，

<a name="19_83"></a>

变异可得故，彼因与果故，

执受主宰故，清净随众故。

<a name="19_84"></a>

初生渐大生，随增所依事，

变异及成熟，如是劣与胜。

<a name="19_85"></a>

光明不光明，行余境具种，

无种之本体，以及影像生。

<a name="19_86"></a>

于此十四生，以因及量别，

增无义非理，所依不容有。

<a name="19_87"></a>

安住不容有，初存终不变，

如是劣与胜，光明不光明。

<a name="19_88"></a>

无行住非理，末者不容故，

随行于心故，有为皆刹那。

<a name="19_89"></a>

大种与六境，说为刹那性，

干涸与增长，性动增减故，

<a name="19_90"></a>

相系彼大故，四种变化故，

色香味及触，同故与彼同。

<a name="19_91"></a>

依柴运行故，渐大可得故，

随心问难故，外亦刹那性。

<a name="19_92"></a>

说人假立有，而非实体有，

不得及颠倒，染法烦恼因。

<a name="19_93"></a>

因有二过故，不说一与异，

诸蕴应成我，彼应成实体。

<a name="19_94"></a>

若说我实有，亦需说必要，

不可说一异，无需非应理。

<a name="19_95"></a>

相世见论言，犹如薪与火，

不可说一异，非理是二故。

<a name="19_96"></a>

有二生识故，彼非缘无义，

故不成见者，直至解脱者。

<a name="19_97"></a>

是主则不令，无常不欲求，

彼需立法相，妨害三菩提。

<a name="19_98"></a>

三过故见等，能作非自生，

亦非彼之缘，无有能见等。

<a name="19_99"></a>

非作无常故，同时恒行故，

见等之能作，自生非应理。

<a name="19_100"></a>

如是住与灭，前无无常故，

亦无三品故，是缘非应理。

<a name="19_101"></a>

诸法无有我，胜义为空性，

缘于我实有，唯一说过失。

<a name="19_102"></a>

染污清净位，断定之分别，

趋入相续别，是以人我说。

<a name="19_103"></a>

我见无需生，串习无始时，

人有则一切，无勤解无解。

<a name="19_104"></a>

如是恒时具，彼德菩萨众，

不舍自利益，亦能成他利。
