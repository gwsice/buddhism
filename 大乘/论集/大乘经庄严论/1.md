# 大乘经庄严论

弥勒菩萨·造论

索达吉堪布·翻译

那莫曼殊师利革玛绕布达雅

梵语：玛哈雅那色札阿朗嘎绕那玛嘎热嘎

藏语：特巴钦波多地坚意夏瓦策里哦雪巴

汉语：大乘经庄严论颂

顶礼一切佛菩萨！<sup>原文中缺少第一品的品名。</sup>

<a name="1_1"></a>

知义讲义语句净，为救苦性悯苦情，

示胜乘法宣行众，圆五义理无上性。

<a name="1_2"></a>

如金打造莲花开，如饥饿人享美食，

如闻佳音启宝箧，彼法此说亦最喜。

<a name="1_3"></a>

如自具德貌美人，佩饰临镜见最喜，

佛法恒具自性德，释义智者此最喜。

<a name="1_4"></a>

如药嗅极臭，品尝则香醇，

法亦住二种，当知义与文。

<a name="1_5"></a>

广深此难法，如王难欢喜，

若行令喜事，能赐胜德财。

<a name="1_6"></a>

具种无价宝，不识人不悦，

如是于此法，愚非智者悦。

## 第二 证成大乘品

<a name="2_1"></a>

未授记俱出，非行境成立，

有无同有无，对治别文故。

<a name="2_2"></a>

诸佛现量眼，亦护诸佛法，

诸时智无碍，是故不应舍。

<a name="2_3"></a>

不全及相违，非方便未说，

故此声闻乘，非谓大乘法。

<a name="2_4"></a>

意乐及宣说，加行与所依，

时违故下劣，彼乘唯下劣。

<a name="2_5"></a>

契入自之经，现调自烦恼，

甚深广大故，不违于法性。

<a name="2_6"></a>

寻思依不定，不遍俗弱者，

凡夫许所依，故彼非彼境。

<a name="2_7"></a>

广大甚深故，成熟无分别，

故此所说二，乃无上方便。

<a name="2_8"></a>

众于非彼处，怖畏受逼恼，

积非福大罪，历经久远时，

无种姓恶友，未积善学修，

由怖畏此法，从此大义堕。

<sup>“众于非彼处……从此大义堕”：此两颂在藏文中为一颂，每句十一字，因字数过多故译成两颂。</sup>

<a name="2_9"></a>

彼外余非有，极深随行故，

种种众多门，恒常而宣说，

非如所说义，佛密意极深，

智者如理观，终不惧此法。

<sup>“彼外余非有……终不惧此法”：此两颂在藏文中为一颂，每句十三字，因字数过多故译成两颂。</sup>

<a name="2_10"></a>

此初依闻已，尔后起作意，

如理作意中，生真义境智，

从中证得法，由法尽生慧，

若无各自证，如何能决定？

<sup>“此初依闻已……如何能决定”：此两颂在藏文中为一颂，每句十三字，因字数过多故译成两颂。</sup>

<a name="2_11"></a>

以自不了知，谓佛非知深，

甚深何以故，非是寻思境？

何故觉深义，是即成解脱？

彼为畏惧处，此者不应理。

<sup>“以自不了知……此者不应理”：此两颂在藏文中为一颂，每句十三字，因字数过多故译成两颂。</sup>

<a name="2_12"></a>

信劣界极劣，劣伴普围绕，

正说深广法，无信成立此。

<a name="2_13"></a>

随闻得具慧，于闻作轻毁，

余种种无量，愚昧何能定？

<a name="2_14"></a>

依文思义时，我慢失智慧，

亦舍善说故，自毁嗔法障。

<a name="2_15"></a>

意罪自性恶，非理色非理，

疑法何须说，等舍善无过。

## 第三 皈依品

<a name="3_1"></a>

遍行承诺证悟胜，以四类性义差别，

为殊胜乘皈三宝，彼谓皈依之第一。

<a name="3_2"></a>

是因初此决定难，彼数千劫修行难，

成就利众义重大，故此大乘皈义胜。

<a name="3_3"></a>

行持救度诸有情，乘智遍行具善巧，

涅槃有寂平等性，当知具慧此遍行。

<a name="3_4"></a>

数喜希求大菩提，无有厌倦真难行，

成佛等同诸佛陀，当知具慧承诺胜。

<a name="3_5"></a>

善生佛陀殊胜子，具心慧资大悲心，

无等种母胎乳母，当知具慧承诺胜。

<a name="3_6"></a>

种种妙相严其身，获得成熟众生力，

得佛无边大乐寂，知救有情大方便。

<a name="3_7"></a>

诸佛大光赐灌顶，真具一切法自在，

知佛眷中示现相，勤以制学治惩摄。

<a name="3_8"></a>

如大臣入波罗蜜，恒视遍大菩提分，

恒正受持三密处，恒不间断利生故。

<a name="3_9"></a>

大福超三界有乐，寂灭大苦胜慧乐，

法蕴胜常身善资，尽习气得解有寂。

<a name="3_10"></a>

具慧善广胜声闻，大义无边恒无尽，

彼善以世出世熟，自在蕴灭亦无尽。

<a name="3_11"></a>

知求彼性誓由悲，遍知利乐难无厌，

出离恒具乘证德，表法性得慧皈胜。

<a name="3_12"></a>

行此皈依具大义，彼增无量功德资，

悲心周遍此众生，亦兴无等大圣法。

## 第四 种姓品

<a name="4_1"></a>

有殊胜法相，相及种姓类，

过患与功德，二喻各四种。

<a name="4_2"></a>

界性及信解，修行异类别，

果异可得故，定说有种姓。

<a name="4_3"></a>

彼善悉超胜，一切及大义，

无尽之因故，是谓种姓胜。

<a name="4_4"></a>

自性与随增，彼为所能依，

当知有无性，功德度脱义。

<a name="4_5"></a>

加行前悲心，信解与安忍，

真实行善法，定说种姓相。

<a name="4_6"></a>

种姓定不定，诸缘夺不夺，

如是若略摄，种姓分四种。

<a name="4_7"></a>

习惑恶友伴，贫困随他转，

简言当了知，种姓过有四。

<a name="4_8"></a>

久远堕恶趣，能速得解脱，

于彼受轻苦，具悲成熟众。

<a name="4_9"></a>

当知如金类，无量善根处，

智慧具无垢，亦是威力处。

<a name="4_10"></a>

当知如至宝，大菩提因故，

大智圣等持，群生利益处。

<a name="4_11"></a>

有终定造罪，有尽灭白法，

无解脱分善，善微远离因。

<a name="4_12"></a>

于为利他说，深广之大法，

未知生大信，修行能安忍，

彼等终究时，胜二圆满者，

随增种姓知，菩萨具性德。

<sup>“于为利他说……菩萨具性德”：此两颂在藏文中为一颂，每句十五字，因字数过多故译成两颂。</sup>

<a name="4_13"></a>

生巨德觉树，得乐灭剧苦，

利乐自他果，胜种如妙根。

## 第五 发心品

<a name="5_1"></a>

喜大加行大，义大出生大，

菩萨之思者，起具二利心。

<a name="5_2"></a>

发心于诸地，信解净增意，

异熟许相异，如是永断障。

<a name="5_3"></a>

彼根本许悲，常思利有情，

于法起信解，求智乃所缘。

<a name="5_4"></a>

欲乐上为行，大戒彼所依，

道障是违品，引发及持受。

<a name="5_5"></a>

福慧乃自性，善增彼功德，

恒行波罗蜜，彼即许出离。

<a name="5_6"></a>

地乃此究竟，各自彼行持，

菩萨之发心，决定当了知。

<a name="5_7"></a>

友力因根力，闻力习善生，

不坚及坚固，说他示发心。

<a name="5_8"></a>

近依正等觉，尽积福慧资，

法无分别智，生故许彼胜。

<a name="5_9"></a>

于法及众生，彼事胜佛陀，

获得平等心，故彼极喜胜。

<a name="5_10"></a>

生大喜悦彼，意乐亦清净，

精通余出离，于此当了知。

<a name="5_11"></a>

信解法种子，般若母中生，

禅定生乐胎，悲育之乳母。

<a name="5_12"></a>

为成十大愿，当知是广大，

长难不厌倦，当知是欢喜。

<a name="5_13"></a>

证悟大菩提，得彼方便智，

当知意乐净，精通余地理。

<a name="5_14"></a>

作意依安立，知彼分别故，

亦不分别彼，当知为出离。

<a name="5_15"></a>

当知此发心，如地如纯金，

亦如上弦月，如火大宝藏。

<a name="5_16"></a>

当知如宝源，如海如金刚，

如须弥药王，亦如至亲友。

<a name="5_17"></a>

当知如摩尼，如日寻香音，

如王如仓库，如路亦如乘。

<a name="5_18"></a>

发心如泉水，如雅音大流，

佛子之发心，宣说如大云。

<a name="5_19"></a>

是故功德富，欢喜真发心。

<a name="5_20"></a>

舍极应发心，思他利得方，

大密见最真，弃乐趋寂灭。

<a name="5_21"></a>

具慧生胜心，即护无边罪，

增二恒善慈，乐喜苦亦喜。

<a name="5_22"></a>

利他尚不惜，身命受疲惫，

岂能以他害，趋入行恶业？

<a name="5_23"></a>

悟法皆如幻，生如入乐园，

或兴或衰时，悉无惑苦惧。

<a name="5_24"></a>

自德利生喜，故意生幻变，

严宴胜处戏，非悲尊不具。

<a name="5_25"></a>

悲尊勤他利，无间亦执乐，

世间他依处，生苦焉能惧？

<a name="5_26"></a>

恒依大慈师，他苦逼恼心，

他事近在前，他劝诚惭愧！

<a name="5_27"></a>

首荷有情担，众尊缓行丑，

自他种种缚，精进当百倍。

## 第六 修行品

<a name="6_1"></a>

处行生果大，许佛子修行，

恒诺大忍大，大义真修行。

<a name="6_1"></a>

自他平等心，或爱他胜己，

他利胜己想，自利即他利。

<a name="6_3"></a>

悲尊为利他，自愿遭剧苦，

世人损恼事，他敌亦极悯。

<a name="6_4"></a>

下中上法性，善说令悦入，

解义成熟善，授住慧解脱，

<a name="6_5"></a>

成就胜功德，生种授记灌，

佛智无上果，利他十三性。

<a name="6_6"></a>

随众无倒说，无傲我所通，

忍调远无尽，佛子胜修行。

<a name="6_7"></a>

欲夫入大怖，喜有倒动乐，

喜寂灭自取，悲尊恒灭取。

<a name="6_8"></a>

愚人勤自乐，常苦不得彼，

坚稳恒利他，行二利涅槃。

<a name="6_9"></a>

佛子行持时，入根异行境，

如是随理词，利生彼现行。

<a name="6_10"></a>

恒罪无自在，具慧不执过，

不欲倒行者，于彼反增悲。

<a name="6_11"></a>

此修胜有众，具足最寂灭，

种种德资增，慈心恒摄生。

## 第七 真如品

<a name="7_1"></a>

非有无同异，无生灭减增，

无净成清净，彼为胜义相。

<a name="7_2"></a>

我见非我相，非恶处相异，

唯二迷乱生，解脱唯迷尽。

<a name="7_3"></a>

唯依迷乱士，常苦不证悟，

痴明苦无苦，法性非彼性。

<a name="7_4"></a>

现入缘生法，众生依他作，

是因不见有，见无黑暗相。

<a name="7_5"></a>

此以胜义理，有寂无少别，

然说行善业，灭生得寂灭。

<a name="7_6"></a>

菩萨善积累，无边福慧资，

思法决定故，悟诠具相境。

<a name="7_7"></a>

证彼唯诠境，正住现唯心，

尔后现量证，法界离二相。

<a name="7_8"></a>

慧知心外无，复证心亦无，

具慧知无二，住无彼法界。

<a name="7_9"></a>

具慧无念智，恒成普平等，

依彼重罪净，如剧毒除毒。

<a name="7_10"></a>

佛说妙法善安立，慧住根本法界中，

念相续知唯分别，坚稳速至德海岸。

## 第八 威力品

<a name="8_1"></a>

生语心善恶，彼处出现量，

神通遍行类，无碍获坚力。

<a name="8_2"></a>

四禅得极净，无分别智摄，

作意遍住中，可成第一力。

<a name="8_3"></a>

圣者天梵住，无比广恒住，

行处真供佛，令众生清净。

<a name="8_4"></a>

成坏及有情，世界视如幻，

如愿而自在，真实示种种。

<a name="8_5"></a>

放光令极苦，恶趣升善道，

撼魔妙高宫，令魔生畏怖。

<a name="8_6"></a>

彼于胜会中，游戏无量定，

工巧生胜化，诸时利有情。

<a name="8_7"></a>

智慧自在故，随意现净土，

无佛传佛号，彼等至佛土。

<a name="8_8"></a>

如翼丰满鸟，能成熟有情，

得佛极赞叹，众生应受语。

<a name="8_9"></a>

六通及三明，八解八胜处，

十遍处等持，无量得坚力。

<a name="8_10"></a>

胜自在具慧，无主置自在，

恒唯喜利他，行有坚如狮。

## 第九 成熟品

<a name="9_1"></a>

喜信寂静悲，忍慧具威力，

不夺具支上，佛子成熟相。

<a name="9_2"></a>

胜友等三勤，竟妙摄胜法，

受持大妙法，悲尊成熟相。

<a name="9_3"></a>

知德心不退，速得定受果，

诚信本师故，佛子成熟相。

<a name="9_4"></a>

极禁断惑念，无碍喜乐善，

佛子除惑故，许真成熟相。

<a name="9_5"></a>

自慈见他苦，劣心尽断除，

胜进众尊身，悲他成熟相。

<a name="9_6"></a>

自坚抉择修，恒耐寒等苦，

胜进喜乐善，许忍成熟相。

<a name="9_7"></a>

报净念闻等，尽知善恶说，

可生大智慧，锐智成熟相。

<a name="9_8"></a>

二善增二界，依能生果胜，

如愿成众尊，威力成熟相。

<a name="9_9"></a>

理析妙法智，恒无魔作障，

得胜驳他宗，不夺成熟相。

<a name="9_10"></a>

积善依堪勤，现喜胜静善，

佛子之具支，真实成熟相。

<a name="9_11"></a>

九相成熟尊，能令他成熟，

法性相续增，恒为众至亲。

<a name="9_12"></a>

如疮可破漏，食可用许熟，

此依于二品，灭受说成熟。

<a name="9_13"></a>

许令离成熟，普熟极成熟，

随善证成熟，恒熟上成熟。

<a name="9_14"></a>

佛子以利心，住此成熟众，

父母友于子，及亲非善住。

<a name="9_15"></a>

慈主利乐行，慈爱他众生，

世人自不爱，何况极爱他？

<a name="9_16"></a>

菩萨身受用，无有不施他，

二益成熟他，等施不餍德。

<a name="9_17"></a>

坚性尤无害，自喜谨安他，

二益熟等流，德转成熟他。

<a name="9_18"></a>

他害作利想，重害尚耐受，

知方便忍害，令其真行善。

<a name="9_19"></a>

佛子依胜勤，成熟有情众，

为他一善心，久劫亦不厌。

<a name="9_20"></a>

自在得无上，他供定摧欲，

令他向胜教，令善真增长。

<a name="9_21"></a>

解真如密义，尽除众生疑，

故彼敬佛法，自他善德增。

<a name="9_22"></a>

菩萨悲利生，善趣及三善，

是以下中上，有际平等化。

## 第十 菩提品

<a name="10_1"></a>

无量百难行，积累无量善，

历经无量时，灭尽无量障。

<a name="10_2"></a>

无有诸障垢，得一切种智，

如开珍宝器，真实现佛陀。

<a name="10_3"></a>

百劳难行积众善，大劫久时尽众障，

摧毁地属细障佛，如开大力珍宝器。

<a name="10_4"></a>

诸法本正觉，法者皆非有，

彼虽白法性，依彼非诠彼。

<a name="10_5"></a>

珍宝法因故，犹如珍宝源，

善庄稼因故，许彼如雨云。

<a name="10_6"></a>

佛具一切法，亦离一切法，

生广大法宝，故如宝法源，

于众降善说，无尽大法雨，

极增白稼因，故彼如雨云。

<sup>“佛具一切法……故彼如雨云”：此两颂在藏文中为一颂，每句十五字，因字数过多故译成两颂。</sup>

<a name="10_7"></a>

佛陀恒普救，所有烦恼聚，

一切罪恶行，及与诸老死。

<a name="10_8"></a>

救护诸损恼，恶趣非方便，

坏聚小乘故，为第一皈处。

<a name="10_9"></a>

许佛皈处无比胜，救离众怖坏聚乘，

恶趣多苦非方便，生死烦恼诸恶途。

<a name="10_10"></a>

佛法圆满身，妙法化情智，

悲众至彼岸，许佛胜皈处。

<a name="10_11"></a>

尽除诸世间，众生之损害，

富足诸圆满，许佛大皈处。

<a name="10_12"></a>

烦恼所知障，种子久恒具，

断除极广大，诸相尽摧毁，

获转依白法，最胜德即佛，

极净无分别，大境智道得。

<sup>“烦恼所知障……大境智道得”：此两颂在藏文中为一颂，每句十五字，因字数过多故译成两颂。</sup>

<a name="10_13"></a>

佛住彼见众，如居山王上，

尚悯乐寂众，况于喜有人？

<a name="10_14"></a>

依入极入非趋入，舍离普入二不二，

平等超胜遍行性，承许如来之转依。

<a name="10_15"></a>

虚空恒遍行，许佛亦如是，

虚空遍行色，佛遍行众生。

<a name="10_16"></a>

犹如水器破，月影不显现，

恶劣有情前，佛影亦不现。

<a name="10_17"></a>

如火依他燃，亦依他熄灭，

如是亦当知，佛陀现不现。

<a name="10_18"></a>

如鼓未擂击，普皆出音声，

如来无造作，普皆出善说。

<a name="10_19"></a>

犹如摩尼珠，无勤显自光，

佛陀无造作，定显诸事业。

<a name="10_20"></a>

犹如虚空中，世事不断现，

如是无漏界，佛业不间断。

<a name="10_21"></a>

犹如虚空中，事恒有坏生，

无漏法界中，佛业有生灭。

<a name="10_22"></a>

前后虽无别，无有诸障垢，

真如许佛陀，非净非不净。

<a name="10_23"></a>

空性即清净，无我得胜我，

佛得净我故，成大本性我。

<a name="10_24"></a>

是故佛非有，亦不说无有，

问佛如是语，许为无记理。

<a name="10_25"></a>

譬如铁热息，眼翳已消除，

佛陀心与智，不说有与无。

<a name="10_26"></a>

无漏界中佛，无身如虚空，

随从前身故，非一亦非多。

<a name="10_27"></a>

力等佛陀法，菩提如宝源。

于众善庄稼，亦许如密云。

<a name="10_28"></a>

福智圆满故，承许如满月。

智慧放光故，承许如大日。

<a name="10_29"></a>

犹如日轮中，无量光融合，

恒常行一事，亦能照世界。

<a name="10_30"></a>

如是无漏界，无量佛融合，

同行一事业，显现智慧光。

<a name="10_31"></a>

如出一日光，是集诸光辉，

当知诸佛智，出生亦如是。

<a name="10_32"></a>

犹如日光辉，照耀无我所，

如是佛陀智，入亦无我所。

<a name="10_33"></a>

如射一日光，能照诸众生，

如是佛陀智，顿现诸所知。

<a name="10_34"></a>

犹如许日光，云等作遮障，

如是佛陀智，众生过遮障。

<a name="10_35"></a>

犹如以结牵，染衣艳不艳，

如是引力牵，众智显不显。

<a name="10_36"></a>

说此无垢界，佛陀之法相，

处事业甚深，空中画图案。

<a name="10_37"></a>

真如于一切，无别然清净，

即是如来故，众生具彼藏。

<a name="10_38"></a>

声闻之财富，胜伏世间财，

辟支佛地者，亦胜声闻财，

<a name="10_39"></a>

彼者之财富，不及菩萨财，

菩萨之财富，不及如来财。

<a name="10_40"></a>

佛陀之财富，为何何如何，

几何何时入，无量不可思。

<a name="10_41"></a>

五根转依时，照了一切境，

生千二百德，获得妙自在。

<a name="10_42"></a>

意者转依时，则随自在行，

无分别智净，获得妙自在。

<a name="10_43"></a>

根识转依时，如愿显受用，

清净佛刹土，获得妙自在。

<a name="10_44"></a>

分别转依时，智慧诸事业，

恒常无阻碍，获得妙自在。

<a name="10_45"></a>

所依转依时，佛陀无垢处，

不住之涅槃，获得妙自在。

<a name="10_46"></a>

欲染转依时，安住佛大乐，

见女无烦恼，获得妙自在。

<a name="10_47"></a>

空想转依时，想事得降临，

行走驾驭色，获得妙自在。

<a name="10_48"></a>

佛陀无垢处，转依无有量，

成办无边事，自在许无量。

<a name="10_49"></a>

世间诸方所，如来之善说，

增善世间者，清净至最胜，

未曾积善者，增善至最胜，

如是未熟熟，恒行非无余。

<sup>“世间诸方所……恒行非无余”：此两颂在藏文中为一颂，每句十一字，因字数过多故译成两颂。</sup>

<a name="10_50"></a>

世间诸方所，坚稳恒诸时，

难得大菩提，胜德具足胜，

常坚成无依，依怙大稀有，

由行妙理故，亦非大稀有。

<sup>“世间诸方所……亦非大稀有”：此两颂在藏文中为一颂，每句十三字，因字数过多故译成两颂。</sup>

<a name="10_51"></a>

佛顿时有前，转数百法轮，

有前不现生，有前异生行，

有前现证觉，有前示涅槃，

彼处中不动，同时行一切。

<sup>“佛顿时有前……同时行一切”：此两颂在藏文中为一颂，每句十五字，因字数过多故译成两颂。</sup>

<a name="10_52"></a>

诸佛无是想：此为我成熟，

此当余今熟，然于诸方所，

恒常以三门，善法成熟众。

<sup>“诸佛无是想……善法成熟众”：此一颂半在藏文中为一颂，每句十五字，因字数过多故译成一颂半。</sup>

<a name="10_53"></a>

如日无勤中，光芒照诸方，

无边众光明，成熟众庄稼，

如是具法日，寂灭法之光，

无边照诸方，成熟诸有情。

<sup>“如日无勤中……成熟诸有情”：此两颂在藏文中为一颂，每句十三字，因字数过多故译成两颂。</sup>

<a name="10_54"></a>

犹如一灯燃众灯，无量无数彼无尽，

一者成熟化众熟，无量无数彼无尽。

<a name="10_55"></a>

如此大海不餍水，百川流入彼不增，

佛界趋入恒不断，不餍不增最稀有。

<a name="10_56"></a>

诸法之真如，二障清净相，

缘性缘彼智，无尽自在相。

<a name="10_57"></a>

知真如一切，由修真实生，

诸时令有情，生二无尽果。

<a name="10_58"></a>

身语意幻化，加行方便事，

等持总持门，具足二无量。

<a name="10_59"></a>

自性法受用，幻化相异入，

此者即名谓，佛陀法界净。

<a name="10_60"></a>

佛陀身类别，自性及报身，

别者即化身，初为二所依。

<a name="10_61"></a>

报身于诸界，摄眷刹名身，

圆满受用法，事业悉相异。

<a name="10_62"></a>

自性身平等。细微与彼系，

随意示受用，许是报身因。

<a name="10_63"></a>

佛无量幻化，承许为化身。

二利圆满者，一切相住二，

<a name="10_64"></a>

恒示工巧生，大菩提涅槃，

佛陀此化身，解脱大方便。

<a name="10_65"></a>

当知以三身，摄集佛陀身，

是说三身者，自他利所依。

<a name="10_66"></a>

所依及密意，事业平等性，

自性不间断，相续恒常性。

<a name="10_67"></a>

镜智不动摇，三智依于彼，

平等妙观察，成所作唯此。

<a name="10_68"></a>

镜智无我所，恒具不偏堕，

所知皆不昧，恒非偏向彼。

<a name="10_69"></a>

诸智之因故，如智大源泉，

即是报身佛，现智影像故。

<a name="10_70"></a>

平等性智者，许修有情净，

不住而寂灭，许为等性智。

<a name="10_71"></a>

彼于一切时，具足大慈悲，

依有情信解，定示佛陀身。

<a name="10_72"></a>

妙观察智者，所知恒无碍，

唯是如等持，总持之宝藏。

<a name="10_73"></a>

于诸眷属中，示现诸自在，

斩断诸怀疑，普降大法雨。

<a name="10_74"></a>

成所作智者，诸界以种种，

无量无边化，成办众生利。

<a name="10_75"></a>

佛陀彼幻化，恒行成作别，

数刹于诸时，当知不可思。

<a name="10_76"></a>

受持平等心，尽宣真实法，

成办所作故，真实生四智。

<a name="10_77"></a>

种姓别有义，圆满无始故，

佛非一净住，无异故非多。

<a name="10_78"></a>

是无真如者，许为有之最，

不缘一切相，许为缘之最。

<a name="10_79"></a>

修行不观相，许为修之最，

无见获得中，亦许得之最。

<a name="10_80"></a>

视为上远相，我之精进胜，

菩萨我慢者，名为远菩提。

<a name="10_81"></a>

如是诸宣说，视为唯分别，

菩萨无分别，名定得菩提。

<a name="10_82"></a>

如河非地下，处异水亦异，

水少作用异，微小水生用。

<a name="10_83"></a>

入海则一切，处一水一大，

用一水群生，恒常大受用。

<a name="10_84"></a>

未入佛坚稳，依异慧亦异，

证小自作异，少众常受用。

<a name="10_85"></a>

入佛则彼等，依一大证一，

事业合为一，大众大受用。

<a name="10_86"></a>

佛具无比白法故，亦是利乐之因故，

安乐善妙无尽源，具慧应受菩提心。

*大乘经庄严论中第十菩提品终*

初成皈种姓，发心自他利，

真如义威力，成熟及菩提。
